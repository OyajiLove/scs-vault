### 1. Buzz Memory Tracker

**Source:** Vol 6 line 9, mentioned as Option 1

**Seed:** "How Buzz creates legacy events and long-term crowd recall"

**Concept:** High Buzz moments don't just affect current momentum, they create permanent memories that fans recall years later. A worker who was white-hot in 2019 might get "remember when" chants in 2024.

**Potential Mechanics:**
- Buzz threshold that triggers Memory Snapshot creation
- Memory types: viral promos, shocking returns, legendary matches
- How these memories affect future crowd reactions
- "Remember when" crowd chants for workers who've cooled off
- Legacy Buzz (distinct from current Buzz, represents historical heat)

**Potential System Connections:**
- [[Buzz System/_Buzz System Index|Buzz System]] - Parent system
- [[Fan Memory Engine/_Fan Memory Index|Fan Memory Engine]] - Storage and recall
- [[Crowd_Memory]] - In-match crowd behavior

### 2. Buzz-to-Push Converter

**Source:** Vol 6 line 9, mentioned as Option 2

**Seed:** "AI/Player system where high Buzz translates into title shots, TV time, or political leverage"

**Concept:** Formal rules for how sustained Buzz creates booking pressure. When a worker is hot, the system should push the AI (or warn the player) to capitalize, and penalize ignoring hot workers.

**Potential Mechanics:**
- Buzz thresholds that unlock booking options (e.g., Buzz 15+ = title shot eligible)
- AI Booker pressure to push high-Buzz workers
- Political leverage: high-Buzz workers can demand better spots
- TV time allocation based on Buzz rankings
- "Wasted Heat" penalty when hot workers are underbooked
- Resentment buildup when Buzz is ignored

**Potential System Connections:**
- [[Buzz System/_Buzz System Index|Buzz System]] - Parent system
- [[Buzz System/Push Resistance|Push Resistance]] - Political friction from ignored Buzz
- AI Booker Logic - Decision-making under Buzz pressure
- Contract Negotiations - High Buzz = leverage

### 4. Buzz-Driven Push Tiers

**Source:** Vol 6 line 14, mentioned as Option 3

- Buzz 17-20: Main event priority, protection booking

**Concept:** Formal system for modeling Popularity decay when workers aren't being pushed, aren't appearing, or have lost momentum.

- Casual vs hardcore fan decay rates (casuals forget faster)
