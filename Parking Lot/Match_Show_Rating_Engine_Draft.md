## MATCH FLAVOUR SYSTEM (NEW)

**Concept:** Booker assigns a **Flavour** to individual matches:
- Technical Showcase
- Blood Feud Brawl
- Hardcore War
- Emotional Storytelling
- High-Flying Spectacle
- Comedy
- etc.

**Key Rules:**
- Flavour ‚â† Match Type (it's a thematic lens within house style)
- Flavour-Style Compatibility matters (technical flavour in deathmatch = bad)
- Worker-Flavour compatibility matters (Malenko in technical match = boost)
- Companies can emphasize different styles within their house style

**Example:** WWE 2025 is entertainment-first but can book a technical showcase between two talented workers by selecting Technical flavour.

## ROOKIE HANDLING SYSTEM (LOCKED)

**Problem with TEW:** Rookies tank match ratings just by existing, even when booked properly.

**SCS Solution:**

| Factor | Handling |
|--------|----------|
| Rookie Overness vs Crowd Expectation | Rookies only hurt if they dominate too much relative to expectations |
| Rookie Spotlight | Too much shine without earning it = fans turn, quality suffers |
| Development Bonus | Proximity to over stars + selling well = boosts hidden learning stats |
| Match Structure Weighting | Fall guys/heat-takers have lower influence on rating unless controlling major spots |
| Heat Segments / Struggle Moments | Booking respects hierarchy = quality holds or improves |
| Learning Tree Memory | Consistent exposure to legends accelerates long-term growth (NJPW Young Lion system) |

**Rookies HELPED when:**
- Booked humbly
- Structured to struggle
- Positioned to eat falls or build sympathy
- Booked as underdogs trying their best

**Rookies HURT when:**
- Allowed to dominate heat
- Control long offensive segments against major stars
- Spotlighted way beyond fan acceptance too early

**Bad Booking Example:**
Rookie X gets long offense stretches against prime Kenny Omega ‚Üí Crowd expectation shattered ‚Üí Match rating drops from booking dissonance

**Good Booking Example:**
Rookie X tags in, fights valiantly, takes big offense, gets pinned clean after 2 minutes ‚Üí Omega shines, rookie gets sympathy ‚Üí Match rating stable, Rookie's "Growth Memory" ticks up

| Factor | Rise | Decline |

## CREATIVE HOT STREAK SYSTEM v1.0 (LOCKED)

**Momentum Tracking:**

| Layer | How It Works |
|-------|-------------|
| Booking Trust Memory | Fans track recent weeks' emotional payoffs, betrayals, title changes, feud peaks |
| Emotional Resonance Curve | If multiple shows escalate crowd investment without collapse, emotional energy builds |
| Narrative Freshness Tracker | Booking must feel varied, creative; not repeating patterns |
| Worker Usage Spread | Over-relying on 1-2 workers without elevating others risks burnout |

**Hot Streak Activation:**

| Trigger | What Happens |
|---------|-------------|
| 5-7 Weeks of High Emotional Resonance | Crowds invest deeper, shows feel "must-watch" |
| Booking Memory Trust Level Reaches Threshold | Fans forgive minor booking mistakes temporarily |
| Media Buzz Activates | Fictional press, newsletters start praising promotion's golden age |
| Worker Morale Rises | Backstage excitement increases, better promos, fewer no-shows |

**Hot Streak Benefits:**

| Layer | Boost |
|-------|-------|
| Natural Crowd Warmth | Higher base pop levels at start of matches |
| Feud Accelerators | Heats rise faster, fans invest in smaller emotional beats |
| Merchandise Spikes | Hot streak eras = merchandise explosions |
| Worker Development Buff | Rookies and midcarders grow faster during hot streaks |
| Booking Memory Buffer | Fans tolerate fumbles for a few weeks longer |
| Legacy Seeds | Big matches have higher chance of becoming legacy memory anchors |

**Hot Streak Collapse Triggers:**

| Collapse Cause | Effect |
|----------------|--------|
| Overexposure of Top Stars | Running Austin/Rock into ground drains novelty |
| Lazy Booking Patterns | Same matches, finishes, segment styles burn out fans |
| Creative Overcomplication | Higher Power Swerves, Russo Booking = audience exhaustion |
| Locker Room Fractures | Workers burned out, backstabbing, promo coherence drops |
| External Scandals | Drug busts, assault cases shatter emotional momentum |
| World Events | Wars, disasters, economic crashes lower audience emotional bandwidth |

**Legacy Aftermath:**

| Result | Memory |
|--------|--------|
| Sustained Hot Streak (6-12+ Months) | Promotion locks in "Golden Era" memory, talked about decades later |
| Brief Hot Streak (2-5 Months) | Feels mythic to hardcore fans, casuals forget unless archived |
| Catastrophic Collapse | Great eras can be retroactively tainted if collapse is scandalous (WCW 2000) |

## ERA-DEPENDENT HEALTH AND LONGEVITY MODIFIERS (LOCKED)

| Era | Health Curve Reality |
|-----|---------------------|
| 1970s-Early 90s | Early wear and tear peaks by early 30s. Severe health risks from minor injuries. Few preventative interventions. |

| Adaptability Skill | Ability to modify style, cut bump-heavy sequences, work smarter |
