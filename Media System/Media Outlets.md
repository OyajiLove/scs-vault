## Overview

Different media outlets have different reach, credibility, and relationships with wrestling. Understanding outlet types is essential for managing public perception, scandal containment, and buzz generation.

---

## Outlet Categories

### Mainstream Press

**Examples:** Local newspapers, network TV news, national magazines

| Characteristic | Description |
|----------------|-------------|
| **Reach** | Very High (general public) |
| **Credibility** | High (trusted by non-fans) |
| **Access** | Limited (press credentials, interviews) |
| **Kayfabe Stance** | Protected (territory era), Broken (modern) |
| **Coverage Focus** | Major events, scandals, cultural moments |

**Coverage Triggers:**
- Major arena sellouts
- Celebrity involvement
- Scandals (violence, drugs, death)
- Cultural crossover moments
- Local human interest stories

### Trade Publications

**Examples:** Pro Wrestling Illustrated, Wrestling Magazine

| Characteristic | Description |
|----------------|-------------|
| **Reach** | Medium (dedicated fans) |
| **Credibility** | Medium (industry-focused) |
| **Access** | Good (industry relationships) |
| **Kayfabe Stance** | Protected (plays along with storylines) |
| **Coverage Focus** | Matches, rankings, profiles |

**Coverage Triggers:**
- Title changes
- Major matches
- Worker profiles
- Federation news

### Dirtsheets (Newsletters)

**Examples:** Wrestling Observer, Pro Wrestling Torch, Figure Four Weekly

| Characteristic | Description |
|----------------|-------------|
| **Reach** | Low to Medium (hardcore fans) |
| **Credibility** | Variable (insider knowledge vs. speculation) |
| **Access** | Insider sources (workers, agents, office) |
| **Kayfabe Stance** | Broken (reports backstage reality) |
| **Coverage Focus** | Backstage news, rumors, ratings, business |

**Coverage Triggers:**
- Backstage incidents
- Contract negotiations
- Creative decisions
- Industry rumors
- Rating analysis

### Internet Wrestling Community (1995+)

**Examples:** Websites, forums, early blogs

| Characteristic | Description |
|----------------|-------------|
| **Reach** | Growing (tech-savvy fans) |
| **Credibility** | Low to Medium (unverified) |
| **Access** | Variable (some sources, much speculation) |
| **Kayfabe Stance** | Broken (discussion of real names, backstage) |
| **Coverage Focus** | Everything, instant reaction |

### Social Media (2010+)

**Examples:** Twitter, Instagram, YouTube, TikTok

| Characteristic | Description |
|----------------|-------------|
| **Reach** | Very High (viral potential) |
| **Credibility** | Very Low to High (depends on source) |
| **Access** | Direct (workers post themselves) |
| **Kayfabe Stance** | Complex (workers play both sides) |
| **Coverage Focus** | Moments, reactions, personal content |

## Journalist Archetypes

Individual journalists can have different relationships with wrestling:

| Archetype | Behavior | Effect |
|-----------|----------|--------|
| **Mark Journalist** | Protects kayfabe, writes in-character | Positive coverage, no scandal exposure |
| **Smart Journalist** | Reports reality, breaks stories | Scandal exposure, insider buzz |
| **Scandal Hunter** | Focuses on controversy | Magnifies negative stories |
| **Industry Ally** | Friendly to promotion | Favorable coverage, scandal suppression |
| **Hostile Critic** | Antagonistic to wrestling | Negative framing, cultural attacks |
