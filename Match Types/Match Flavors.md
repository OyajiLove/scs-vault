## Overview

Match Flavors define HOW a match is worked, distinct from Match Types (structural format) and House Styles (promotion philosophy). A singles match can be worked as Technical, Brawl, Story, Comedy, etc. Flavor selection affects crowd reaction, worker fit, memory resonance, and promotion identity alignment.

**Critical Distinction:**
- **Match Type:** Structure (Singles, Tag, Cage)
- **Match Flavor:** Work style (Technical, Brawl, Comedy)
- **House Style:** Promotion philosophy (King's Road, Strong Style)

---

## The 14 Match Flavors

| Flavor | Description | Crowd Appeal |
|--------|-------------|--------------|
| **Technical** | Chain wrestling, holds, counters, mat work. Cerebral. | Purists, workrate fans |
| **Brawl** | Punches, strikes, fighting. Street fight energy. | Casual fans, blood feuders |
| **Story** | Psychology-driven, emotional arc, selling the narrative | All audiences when done well |
| **Spotfest** | High spot density, big moves, less selling between | Casual fans, highlight reel seekers |
| **Aerial** | High-flying, dives, acrobatics | Junior/cruiserweight crowds |
| **Hardcore** | Weapons, blood, brawling outside ring | Hardcore loyalists |
| **Deathmatch** | Extreme violence, glass, fire, barbed wire | Niche underground |
| **Comedy** | Humor, gags, entertainment over competition | Family audiences, comedy fans |
| **Epic** | Long, dramatic, escalating. Fighting spirit. | Serious fans, special occasions |
| **Sprint** | Fast, short, intense. High impact, quick finish. | TV audiences, pacing tool |
| **Old-School** | Classic structure, traditional psychology | Traditionalist fans, older demos |
| **Glamour** | Presentation-focused, entrance-heavy, spectacle | Casual fans, event atmosphere |
| **Sports Ent** | WWE-style presentation, story over work | Mainstream audiences |
| **Shoot** | Worked-shoot style, MMA influence, stiff | Shoot-style fans, credibility seekers |

## Road Agent Flavor Guidance

When Road Agents plan matches, they must consider:

1. **Match Type locked** (by booker)
2. **Flavor compatibility** (from chart above)
3. **Worker specialties** (who can deliver)
4. **Promotion identity** (does flavor fit house style?)
5. **Show pacing** (flavor variety across card)
6. **Crowd emotional target** (what feeling to create)

**Example:**
- Booker assigns: Cage Match
- Agent checks: Cage + Brawl = âœ…, Cage + Comedy = ðŸ”´
- Agent considers workers: Both are brawlers = good fit
- Agent decides: Brawl flavor with Story elements for feud payoff
