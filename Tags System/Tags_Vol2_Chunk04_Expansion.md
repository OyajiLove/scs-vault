## NON-BINARY TAG STRENGTH SYSTEM

ðŸ“Œ **Source:** Vol 2 Chunk 04 (CRITICAL system mechanic)

Tags are no longer binary (has/doesn't have). Each tag has a **percentage strength** from 0-100%:

| Strength % | Meaning |
|------------|---------|
| 0% | Doesn't have the trait at all |
| 1-30% | Very minor influence, rarely triggers |
| 31-60% | Moderate, noticeable but secondary |
| 61-90% | Strong, regular influence on booking/crowd |
| 91-100% | Defining trait, core identity |

### Example: Kevin Nash

| Tag | Estimated Strength |
|-----|-------------------|
| Cool Heel | 95% |
| Storyteller (Shoot) | 90% |
| Big Match Player | 80% |
| Is This Mania? | 85% |
| Political Kingmaker | 70% |

### Career Evolution

Tag percentages can grow or decay over time:
- Young Roman Reigns: Cool Heel 30%, Big Match Player 40%
- Modern Roman Reigns: Cool Heel 80%, Big Match Player 90%

## ETHNIC FOLK HERO SYSTEM

ðŸ“Œ **Source:** Vol 2 Chunk 04 (regional cultural mechanics)

### System Requirements

1. **Worker Ethnicity:** Each worker has defined ethnicity in database
2. **Regional Demographics:** Regions have demographic profiles (% Italian, % Black, % Hispanic, etc.)
3. **Cross-Check:** Ethnic Folk Hero tag triggers bonus effects only in matching regions

### Effects

| Condition | Effect |
|-----------|--------|
| Matching Ethnicity + Region | Big loyalty boost, harder heel turns, longer drawing power |
| Mismatched Ethnicity | No automatic boost (unless Grassroots Hero Build earned) |
| Changing Generations | Regions evolve demographics over decades |
| Overexposure/Betrayal | Extra severe backlash for Folk Heroes |

### Historical Examples

| Worker | Ethnicity | Region | Effect |
|--------|-----------|--------|--------|
| JYD | Black | Mid-South | Huge loyalty from Black audiences |
| Sammartino | Italian | Northeast | Massive Italian-American fan surge |
| Pedro Morales | Puerto Rican | NYC | Strong Puerto Rican cultural bond |

### Grassroots Hero Build

Workers can earn Folk Hero-like loyalty over time through consistent performance and emotional connection, even without ethnic ties to the region.
