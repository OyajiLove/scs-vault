## Overview

> "We're not building characters. We're building how people remember them."

Comic-style renders as the official visual identity of the base world. They support gimmick-first design, capture archetypes and exaggeration vital to drama-driven sim, and embrace worker memory, drift, and rebirth as visual myth.

---

## Format & File Details

| Element | Guideline |
|---------|-----------|
| Format | PNG with transparent background |
| Framing | Waist-up by default (mid-torso to head) |
| Backup Option | Shoulders-up for masked workers, promos, or legacy portraits |
| Resolution | Minimum 512x512, scalable for UI (cards, profiles, banners) |
| Lighting | Studio neutral or stylized soft shadows, mild contrast |
| Export Cropping | Centered with 5-10% padding for overlays |

---

## Tone & Emotion

Style should read like a comic cover, not a panel:

| Attribute | Description |
|-----------|-------------|
| Expression | Strong, readable emotion: defiance, pride, menace, serenity, anguish |
| Pose | Slight posture variation, shoulders relaxed or coiled |
| Eyes | Critical. Gimmick memory lives in the stare. Highlight intensity or emptiness. |
| Emotion Layer | Design like telling audience: "Remember this face." |

---

## Visual Cohesion Guidelines

| Design Pillar | Guideline |
|---------------|-----------|
| Colour Grading | 1-2 dominant tones per render; avoid rainbow unless gimmick demands |
| Linework | Bold, clean comic-style shading. Graphic novel cover, not painterly. |
| Detail Level | High enough for scars, makeup, stitching; low enough to not clash with UI |
| No Background Art | All characters isolated (transparent PNGs); effects added dynamically in-game |

---

## Tag-Driven Stylization

Apply minor variants based on in-universe tags:

| Tag | Render Quirk |
|-----|--------------|
| Immortal / Legacy | Desaturated edges, faded vignette |
| Masked / Supernatural | Sharper contrast, unnatural glow in lighting |
| Modern Pop Gimmick | Bright tones, slight lens flare on shades or jewels |
| Grit-Based Fighter | Harsh shadows, neutral tones, chipped gear |

---

## Visual Memory Principles

Every render should imply:
- **History** (what this character's been through)
- **Emotion** (what they carry)
- **Aura** (what they mean to the crowd)

Even if the player's never seen them wrestle.

## Visual Anchor Layers v2.0

Modular identity system tied to character identity and shape language, not just pose/position.

### Layer Categories

| Layer | Function | Notes |
|-------|----------|-------|
| Craniofacial Structure | Skull shape, jawline, brow, nose | Defines the "face silhouette" (base anchor) |
| Eye Set & Expression | Distance between eyes, eye shape, eyebrow arch | Key to continuity, mood and emotional tone |
| Mouth & Chin Region | Lip fullness, expression lines, dimples, chin cleft | High emotional granularity |
| Hair Base | Hairline, density, growth direction | Differentiates characters at a glance |
| Skin Tone + Texture Map | Base complexion, pore/freckle distribution, weathering | Ensures overlays blend naturally |
| Body Frame Anchor | Shoulder width, trapezius slope, neck length | Preserves physical silhouette |
| Tag-Based Detail Anchors | E.g., "Damaged Jawline," "Iconic Eyebrows," "Burn Scar Left Side" | For workers with iconic physical quirks |
