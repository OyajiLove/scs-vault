## Design Note

Morale is currently **implicit** rather than explicit. It's expressed through:

1. **Hidden Personality:** Drive affects motivation, Ego affects demands
2. **Contract System:** Contract satisfaction = de facto morale
3. **Injury System:** Physical state affects mental state
4. **Faction System:** Faction stability affects member morale

A standalone Morale System could consolidate these, but may create redundancy.
