## Relationship Types (#178)

### Primary Relationship Categories

| Type | Description | Strength Range |
|------|-------------|----------------|
| **Family** | Blood relatives, marriage ties | Strongest loyalty, hardest to break |
| **Mentor/Student** | Teaching relationship, career guidance | Development bonuses, loyalty |
| **Friends** | Personal bonds beyond professional | Morale support, locker room politics |
| **Romantic** | Intimate relationship | Complicated, can be positive or negative |
| **Rivals** | Professional respect with competition | May be friendly or hostile |
| **Enemies** | Active antagonism | Sabotage potential, real heat |
| **Professional** | Work relationship only | Neutral, functional |
| **Former [Type]** | Past relationship that changed | History remains, tension possible |

### Relationship Strength Scale

| Level | Description | Behavior |
|-------|-------------|----------|
| **Absolute (95-100)** | Unbreakable bond | Would follow anywhere, sacrifice career |
| **Strong (75-94)** | Deep connection | Major loyalty, strong support |
| **Moderate (50-74)** | Solid relationship | Professional respect, some personal |
| **Weak (25-49)** | Surface level | Functional but fragile |
| **Strained (1-24)** | Tension present | May break, avoid booking together |
| **Hostile (Negative)** | Active conflict | Heat, sabotage, political warfare |

## Relationship Timeline System (#177)

### Temporal Evolution

Relationships are not static. They have:

| Component | Description |
|-----------|-------------|
| **Start Date** | When relationship began |
| **Current Status** | Active strength and type |
| **Historical Breakpoints** | Major events that shaped relationship |
| **Evolution Checks** | Periodic assessments for change |

### Evolution Check Triggers

| Trigger | Frequency | Effect |
|---------|-----------|--------|
| **Time Passage** | Every 6-12 months | Gradual drift possible |
| **Scandal Involving One Party** | When scandal breaks | Trust test |
| **Betrayal Angle** | When booked | Relationship stress |
| **Major Match** | Big event together | Can strengthen or strain |
| **Faction Changes** | When alignment shifts | Loyalty test |
| **Contract Disputes** | When negotiating | Competing interests |

### Evolution Outcomes

| Outcome | Description |
|---------|-------------|
| **Strengthen** | Shared experiences build bond |
| **Maintain** | Status quo preserved |
| **Drift Apart** | Distance grows, strength decreases |
| **Transform** | Relationship type changes (friend ‚Üí enemy) |
| **Collapse** | Relationship ends, often badly |

## Relationship Web Hidden System

### Core Concept

Every worker has a hidden network of relationships that affects their behavior and responses to booking decisions.

### Network Effects

| Scenario | Effect |
|----------|--------|
| **Hiring a friend** | Morale boost for both, easier chemistry |
| **Firing a friend** | Morale damage, potential political fallout |
| **Booking against enemy** | Real tension, can enhance or derail match |
| **Romantic complications** | Affects both parties, potential scandal |
| **Mentor leaving** | Student morale hit, development impact |
| **Family member booked badly** | Family-wide resentment |

### Relationship Discovery

Players learn about relationships through:

| Method | Information Revealed |
|--------|---------------------|
| **Backstage reports** | "Worker A and B seem close" |
| **Faction formation** | Natural groupings reveal bonds |
| **Contract negotiations** | "I want to work with my friend" |
| **Booking feedback** | Chemistry notes in match reports |
| **Scandal fallout** | Who defends whom |
| **Departure patterns** | Who follows whom out the door |

## Romantic Relationship Complications (#179)

### Romance States

| State | Description |
|-------|-------------|
| **Developing** | Interest forming, not yet official |
| **Active (Stable)** | Healthy relationship |
| **Active (Troubled)** | Issues present, breakup possible |
| **Broken Up** | Ended, aftermath depends on terms |
| **Married** | Strongest bond, package deal tendencies |

### Romantic Scenario Effects

| Scenario | Effect |
|----------|--------|
| **In-roster romance (stable)** | Minor morale boost, package deal tendencies |
| **Romance breakup (amicable)** | Minor awkwardness, manageable |
| **Romance breakup (hostile)** | Potential scandal, morale crash, one may leave |
| **Love triangle** | Major drama, split locker room, angles or chaos |
| **Public cheating scandal** | PR nightmare, both parties affected, career damage |
| **Cross-promotion romance** | Loyalty complications, contract leverage |

### Historical Examples

| Case | Type | Outcome |
|------|------|---------|
| Edge/Lita/Matt Hardy | Love triangle | Real heat, worked into angle, long-term fallout |
| Randy Savage/Elizabeth | Married | Package deal, storyline goldmine, protective issues |
| Triple H/Stephanie | Power couple | Political dynasty, industry-shaping |
| Paige/Del Rio | Cross-promotion | Contract complications, scandal spiral |

## Family Dynamics

### Family Relationship Strength

Family bonds are the strongest and hardest to break:

| Factor | Effect |
|--------|--------|
| **Blood relation** | Near-unbreakable baseline |
| **Marriage** | Strong but can end |
| **In-laws** | Moderate, depends on marriage health |
| **Wrestling dynasties** | Compound loyalty across generations |

### Family Booking Considerations

| Scenario | Consideration |
|----------|---------------|
| **Family vs. family** | May refuse, or bring real emotion |
| **Family member booked poorly** | Entire family resentment |
| **One family member fired** | Others may leave |
| **Family stable formation** | Natural chemistry, loyalty bonus |

### Dynasty Examples

| Family | Impact |
|--------|--------|
| **Hart Family** | Complex loyalties, multiple generations |
| **Anoa'i Family** | Massive network, political power |
| **McMahon Family** | Industry control, family drama = company drama |
| **Guerrero Family** | Lucha dynasty, cross-border impact |

## Mentor/Student Relationships

### Formation

| Requirement | Description |
|-------------|-------------|
| **Experience gap** | Mentor must have significant seniority |
| **Skill relevance** | Mentor must have teachable knowledge |
| **Personality match** | Some archetypes don't mentor well |
| **Opportunity** | Must work together, share time |

### Benefits

| Benefit | Description |
|---------|-------------|
| **Skill transfer** | Student learns faster with good mentor |
| **Tag transfer** | Student may acquire mentor's style tags |
| **Political protection** | Mentor's standing helps student |
| **Career guidance** | Mentor advocates for student's push |

### Complications

| Issue | Effect |
|-------|--------|
| **Student surpasses mentor** | Ego clash possible |
| **Mentor's decline** | Student may distance |
| **Betrayal angle** | Creates nuclear heat if relationship was strong |
| **Mentor retirement** | Student inherits legacy pressure |
