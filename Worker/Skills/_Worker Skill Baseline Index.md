## USAGE

When creating or validating worker stats:

1. Find comparable worker in calibration files
2. Use as baseline, adjust for era/context
3. If unsure, err toward 10-14 range (solid but not exceptional)
4. Reserve 18-20 for truly legendary ability in that area
5. Don't be afraid of low scores (3-8) for genuine weaknesses
