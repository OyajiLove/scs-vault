### Key Corrections

- Umaga (6'4") is NOT a Giant, fits Wildman Brawler better
- Yokozuna = honorary giant (bulk over height)
- Luke Gallows was never a technician
- Technical Giant is extremely rare; most giants rely on presence over skill

### Early Giant Baba Note

Best example of Technical Giant. Would go hold-for-hold 60 minutes with The Destroyer. Great mat wrestler in 60s/early 70s.

## Hardcore Archetype (#17, #223)

**Core Skills (MANDATORY):**

| Skill | Minimum |
|-------|---------|
| **Brawling** | 12+ |
| **Believability** | 12+ |

**CRITICAL DESIGN DECISION:** NO separate "Hardcore" skill rating. Hardcore performance is emergent from brawling, believability, durability, psychology, menace, tags, and health engine.

### Why No Hardcore Stat?

| Reason | Explanation |
|--------|-------------|
| Redundancy | Real hardcore = storytelling + absorbing punishment |
| Game Balance | Avoid skill-bloat |
| Flexibility | New Jack vs Onita feel totally different |
| Era-Dependent | 1980s hardcore barely existed, 2000s is its own scene |

### Hardcore Subtypes

| Subtype | Description | Examples |
|---------|-------------|----------|
| **Hardcore Icon** | Legendary name draws based on reputation alone | Foley, Onita, Terry Funk |
| **Hardcore Daredevil** | Flying madmen, bump crazy | Sabu, Darby Allin |
| **Garbage Brawler** | Wild weapon brawlers, chaos over finesse | New Jack, Abdullah |
| **Technical Hardcore Hybrid** | Legit skilled wrestlers who choose violence | Cactus Jack '95-97, Masato Tanaka |
| **Violent Psychopath** | Sheer menace and willingness to hurt | Necro Butcher |

- **Injury Risk:** Far higher long-term
- **Fan Loyalty:** Higher among hardcore crowds
- **Mainstream Push:** Harder unless reworked
- **Decline Curve:** Fall off younger due to accumulated damage

## Scar Tissue Tag (#224)

**Description:** Worker's body shows heavy scarring from previous battles.

**Effects:**

| Effect | Description |
|--------|-------------|
| Menace | +1 to +3 based on existing menace score |
| Marketability | Slight penalty for "clean image" promotions |
| Hardcore Connection | Slight bonus with hardcore audiences |
| Match Ratings | Small bonus in hardcore/deathmatch events if visible |
| Dynamic Acquisition | Can appear during career from accumulated damage |
| Promo Material | "Look at these scars, brother, this ain't makeup!" |

**Examples:** Onita, Foley (missing ear), Abdullah (forehead roadmap), Necro, Jun Kasai
