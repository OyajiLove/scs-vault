## Overview

When workers can no longer compete at their previous level, they may transition to non-wrestling roles within the industry. This keeps experienced talent in the system while freeing roster spots for younger workers. Role evolution creates mentorship opportunities, booking depth, and realistic career endpoints.

---

## Available Post-Wrestling Roles

### On-Screen Roles

| Role | Requirements | Function |
|------|--------------|----------|
| **Manager** | Charisma 14+, Character Work 12+ | Accompanies workers, promos |
| **Commentator** | Charisma 15+, Wrestling Knowledge | Calls matches |
| **Authority Figure** | Charisma 12+, any experience | GM/Commissioner character |
| **Interviewer** | Charisma 12+, likeable | Conducts promos/interviews |
| **Special Enforcer** | Presence 14+, credibility | Ringside presence for big matches |

### Behind-the-Scenes Roles

| Role | Requirements | Function |
|------|--------------|----------|
| **Road Agent** | Psychology 16+, 15+ years experience | Lays out matches |
| **Trainer** | Technical 14+ OR Psychology 15+ | Develops talent |
| **Booker** | Psychology 16+, Politics navigation | Plans shows/feuds |
| **Scout** | Industry connections, travel willingness | Finds talent |
| **Producer** | TV experience, organizational skills | Runs production |

## Transition Triggers

### Voluntary Transition

| Trigger | Description |
|---------|-------------|
| **Age Decision** | Worker chooses to step back |
| **Injury Wisdom** | One injury too many, smart exit |
| **Role Opportunity** | Offered attractive backstage position |
| **Family Priority** | Reduces schedule for personal life |
| **Passion Shift** | More interested in training/booking |

### Forced Transition

| Trigger | Description |
|---------|-------------|
| **Physical Inability** | Body can no longer perform |
| **Performance Decline** | Match quality unsalvageable |
| **Contract Decision** | Not renewed as active wrestler |
| **Medical Retirement** | Cleared to work, not wrestle |
| **Promotional Pressure** | Asked to move to backstage |

## Transition Process

### Phase 1: Reduced Schedule

| Element | Description |
|---------|-------------|
| **Part-Time Wrestling** | Fewer matches, bigger moments |
| **Backstage Learning** | Shadow agents, sit in on booking |
| **Test Runs** | Trial commentary, one-off managing |
| **Skill Assessment** | Evaluate fit for different roles |

### Phase 2: Dual Role

| Element | Description |
|---------|-------------|
| **Occasional Matches** | Special attractions only |
| **Primary Backstage** | Main role is non-wrestling |
| **Knowledge Transfer** | Actively mentoring replacements |
| **Legacy Positioning** | Building post-wrestling identity |

### Phase 3: Full Transition

| Element | Description |
|---------|-------------|
| **Retirement Match** | Final in-ring appearance |
| **Full-Time Role** | Exclusively backstage or on-screen non-wrestler |
| **Occasional Returns** | Royal Rumble surprises, legends matches |
| **Legacy Status** | Remembered as worker, known for current role |

## Personality Effects on Transition

| Trait | Effect on Transition |
|-------|---------------------|
| **High Ego** | Resists stepping back, wants spotlight |
| **Low Ego** | Graceful transition, embraces new role |
| **High Loyalty** | Stays with promotion in any capacity |
| **Low Loyalty** | May leave industry entirely |
| **High Traditionalism** | Values training/booking tradition |
| **High Drive** | Throws self into new role |
| **Backstage Politician** | Seeks power positions (booker, executive) |
| **Martyr Spirit** | Excellent trainer, selfless mentor |

## Role Success Factors

| Factor | Effect |
|--------|--------|
| **Respect from Locker Room** | Trainees listen, agents trusted |
| **Promotional Support** | Given real responsibility |
| **Skill Match** | Right person for right role |
| **Letting Go** | Accepts no longer the star |
| **Continued Learning** | Adapts to new responsibilities |

### Failure Modes

| Problem | Result |
|---------|--------|
| **Can't Let Go** | Keeps inserting self in angles |
| **Resentment** | Sabotages those with spots they want |
| **Skill Mismatch** | Bad trainer, bad booker |
| **Politics** | Uses position for vendettas |
| **Disconnection** | Loses touch with current product |
