## Core DNA Attributes

### Demographics

| Attribute | Description |
|-----------|-------------|
| Ethnicity Split | Racial/ethnic composition (affects scandal sensitivity, loyalty to certain worker types) |
| Religious Split | Christian, Shinto, Buddhist, Muslim, Secular, etc. (affects risqu√© tolerance, gimmick reception) |
| Political Leaning | Conservative, Liberal, Socialist, Nationalist, Apolitical (shapes scandal fallout, hero/heel dynamics) |
| Economic Class | Working class, Middle class, Upper class proportions (ticket sensitivity, "hero" archetype connection) |
| Age Skew | Young, Middle-aged, Elderly heavy (scandal reaction, innovation tolerance) |
| Gender Balance | Male-heavy, Female-heavy, Balanced (scandal impact, story preferences) |
| Education Level | High school, College, University rates (storytelling sophistication tolerance) |

### Cultural Values Index

Weighted balance of core fan expectations:

| Value | Description |
|-------|-------------|
| Loyalty | Faithfulness to workers/promotions through hardship |
| Family | Importance of family storylines, legacy, bloodlines |
| Freedom | Rebellion, anti-authority themes |
| Honor | Dignity, sportsmanship, earning victories |
| Toughness | Physical resilience, fighting through pain |
| Rebellion | Anti-establishment, breaking rules |
| Nostalgia | Clinging to past glories, legends, old styles |
| Respect for Authority | Deference to champions, veterans, institutions |
| Defiance | Challenging power structures, underdogs |

### Tolerance Levels

| Attribute | Description |
|-----------|-------------|
| Violence Tolerance | Appetite for blood, hardcore, deathmatches (High/Medium/Low) |
| Risqu√© Tolerance | Sexual content, edgy humor, political commentary (High/Medium/Low) |
| Scandal Sensitivity | How easily fans abandon promotions/workers after scandals (High/Medium/Low) |
| Kayfabe Loyalty | How seriously they treat "the illusion" (High/Medium/Low) |
| Hero Worship Tendency | Need for "pure" heroes vs complex anti-heroes (High/Medium/Low) |
| Nostalgia Pull | Clinging to legends, old styles, past glories (High/Medium/Low) |

### Language Profile

| Attribute | Description |
|-----------|-------------|
| Primary Language(s) | Main language(s) spoken by population |
| Secondary Languages | Minority/immigrant languages with cultural weight |
| Foreign Language Tolerance | Receptiveness to promos in non-majority language |
| Language Pride/Sensitivity | How touchy about language identity (Quebec French, etc.) |

## DNA Evolution Over Time

| Era | Natural Evolution |
|-----|-------------------|
| 70s-80s | Kayfabe loyalty very high; scandals invisible; slow regional storytelling |
| Early 90s | Rise of anti-heroes; hardcore violence tolerance spikes; casual fanbase grows |
| Late 90s | Extreme kayfabe loyalty crash; fans jaded; cult worship grows |
| 2000s | Divided: casuals vs workrate nerds |
| 2010s | Rising progressive fanbases, scandal sensitivity grows, nostalgia revival |
| 2020s | Full fragmentation: casual sports entertainment, hardcore indie loyalists, progressive activists, nostalgic oldheads coexist uneasily |

## Promotion vs Population DNA Interaction

### Recruitment Strength

```
Does Promotion DNA Align With Population?
    ‚Üí Yes ‚Üí Rapid growth, mainstream acceptance
    ‚Üí No ‚Üí Slow niche growth, possible cult loyalty, or stagnation
```

### Cultural Clash

- **Alignment:** Easy growth, mainstream acceptance
- **Smart Clash:** (ECW vs 90s moral panic) Builds cult loyalty
- **Stupid Clash:** (PG WWE trying Attitude revival in 2020s) Collapse or flailing

### Mutation Events

| Event | Mutation Effect |
|-------|-----------------|
| Major Scandal Mishandled | Fanbase fragments, becomes more niche |
| New Star With Different Appeal | Fanbase shifts toward new demographic |
| Regional/Era Shift | Promotions must adapt or fade |
| Booking Style Changes | DNA realigns or alienates old loyalists |
