## Overview

The Excursion System sends developing workers to foreign promotions for extended learning tours. This exposes trainees to different wrestling styles, cultures, and booking philosophies before they return to their home promotion. Excursions are most common in Japanese promotions but can apply to any territory-based system.

---

## Excursion Purpose

| Goal | Description |
|------|-------------|
| **Style Diversification** | Learn techniques not available at home |
| **Character Development** | Develop personality away from home pressure |
| **Ring Time** | Accumulate match experience |
| **Toughening** | Face unfamiliar opponents and conditions |
| **Mystique Building** | Return as "transformed" worker |
| **Skill Polish** | Refine fundamentals in different context |

---

## Excursion Destinations

### By Home Promotion Style

| Home Style | Recommended Destinations | Skills Gained |
|------------|-------------------------|---------------|
| **Japanese Strong Style** | US (storytelling), Mexico (aerial) | Entertainment, Aerial |
| **Lucha Libre** | Japan (stiffness), US (psychology) | Striking, Ring Psychology |
| **American TV Style** | Japan (work rate), UK (catch) | Technical, Stamina |
| **British Catch** | Mexico (speed), Japan (drama) | Aerial, Selling |
| **Deathmatch** | Anywhere traditional | Fundamental repair |

### Destination Effects

| Region | Primary Benefit | Secondary Benefit | Risk |
|--------|-----------------|-------------------|------|
| **Japan** | Technical excellence, stamina | Stiff style adaptation | Culture shock, burnout |
| **Mexico** | Aerial skill, mask tradition | Crowd connection | Injury from high-flying |
| **US (Territories)** | Promo ability, psychology | Travel toughening | Bad habits from carny veterans |
| **US (Major)** | TV production, entertainment | Character work | Getting lost in shuffle |
| **UK** | Technical wrestling, catch style | Strong fundamentals | Low pay, hard conditions |
| **Europe** | Style versatility | International exposure | Language barriers |

## Excursion Duration

| Duration | Type | Effect |
|----------|------|--------|
| **Short (3-6 months)** | Exposure tour | +1 to one skill, minimal style change |
| **Standard (6-12 months)** | Learning tour | +2 to skills, potential tag acquisition |
| **Extended (1-2 years)** | Full development | +3 to skills, multiple tags possible |
| **Long-term (2+ years)** | Deep immersion | Character transformation, new identity |

### Duration Risks

| Risk | Trigger | Effect |
|------|---------|--------|
| **Forgetting Home Style** | 18+ months away | May need retraining on return |
| **Going Native** | Extended stay, success abroad | May not want to return |
| **Injury Abroad** | Any duration | Development interrupted |
| **Lost to System** | Poor tracking | Worker falls through cracks |

---

## Return Dynamics

### Successful Excursion Return

| Element | Effect |
|---------|--------|
| **Mystique Bonus** | +10 to +20 initial pop on return |
| **"Transformed" Perception** | Crowd sees them as leveled-up |
| **New Move Set** | Incorporates foreign techniques |
| **Fresh Character** | May have new look, attitude |
| **Main Roster Ready** | Graduates from Young Lion |

### Failed Excursion

| Outcome | Cause | Effect |
|---------|-------|--------|
| **No Improvement** | Poor destination match | Wasted time, lateral move |
| **Injury Return** | Accident abroad | Development setback |
| **Attitude Problems** | Bad influences | Personality trait shifts |
| **Contract Poached** | Foreign promotion signs them | Lost to home fed |

---

## Booking During Excursion

Workers on excursion are booked by the destination promotion:

| Booking Level | Description | Learning Value |
|---------------|-------------|----------------|
| **Enhancement Talent** | Loses to build local stars | High (learns from better workers) |
| **Midcard** | Competitive matches | Medium-High (genuine experience) |
| **Featured** | Wins, holds secondary titles | Medium (may develop bad habits) |
| **Main Event** | Top of card abroad | Low-Medium (may not want to return) |

**Ideal Excursion:** Enhancement to midcard transition over 6-12 months
