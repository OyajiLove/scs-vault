## Overview

Regime Change occurs when a promotion's leadership changes: new owner, new booker, or new head of creative. These transitions trigger loyalty reassessments, political realignments, and potential instability. Workers who thrived under the old regime may struggle under the new one, and vice versa.

---

## Regime Change Types

### Ownership Change

| Type | Trigger | Stability Impact |
|------|---------|------------------|
| **Inheritance** | Owner death, family succession | Moderate (legitimacy questions) |
| **Sale** | Promotion sold to new owner | High (new direction possible) |
| **Hostile Takeover** | Acquisition against current ownership will | Very High (resistance likely) |
| **Bankruptcy** | Financial collapse, new ownership | Critical (survival mode) |
| **Merger** | Promotions combine | Very High (culture clash) |

### Booker Change

| Type | Trigger | Stability Impact |
|------|---------|------------------|
| **Planned Succession** | Booker retires, chosen successor | Low (continuity) |
| **Firing** | Booker dismissed | Moderate (direction change) |
| **Resignation** | Booker leaves voluntarily | Moderate (talent may follow) |
| **External Hire** | Outsider brought in | High (new philosophy) |
| **Promotion from Within** | Worker becomes booker | Variable (friends/enemies) |

## Worker Response to Regime Change

### Loyalty Reassessment

Every worker evaluates their position under the new regime:

```
Survival Probability = 
  (Relationship with New Regime √ó 0.35) +
  (Value to Promotion √ó 0.30) +
  (Political Skill √ó 0.20) +
  (Contract Security √ó 0.15)
```

### Response Categories

| Response | Trigger | Behavior |
|----------|---------|----------|
| **Embrace** | Good relationship with new regime | Publicly supports, gains favor |
| **Adapt** | Neutral relationship, values survival | Adjusts behavior, waits to see |
| **Resist** | Loyal to old regime, dislikes new | Passive resistance, may sabotage |
| **Exit** | No future under new regime | Seeks departure, may trigger exodus |

### Personality Effects on Response

| Archetype | Typical Response |
|-----------|------------------|
| **Backstage Politician** | Adapts quickly, ingratiates new power |
| **Loyal Soldier** | Struggles if loyal to old regime |
| **Mercenary Soul** | Embraces whoever pays |
| **Ace Complex** | Demands assurance of position |
| **Bombthrower** | May resist publicly |
| **Innocent Idealist** | Evaluates philosophy alignment |

## New Regime Actions

### Common First Moves

| Action | Purpose | Risk |
|--------|---------|------|
| **Clean house** | Remove old loyalists | Exodus trigger |
| **Promote loyalists** | Install own people | Resentment from others |
| **Continuity pledge** | Calm uncertainty | May limit options |
| **Meet with roster** | Assess attitudes | Information gathering |
| **Make example** | Establish authority | Fear vs. respect |
| **Big signing** | Signal new direction | Destabilizes existing roster |

### Booking Philosophy Shift

| Element | Old Regime | New Regime | Impact |
|---------|------------|------------|--------|
| **Main eventers** | Established stars | New favorites | Major upheaval |
| **Style preference** | Traditional | Modern (or vice versa) | Skill revaluation |
| **Push targets** | Veterans | Young talent | Generational tension |
| **Finish philosophy** | Clean finishes | Screwjob endings | Crowd adjustment |

## Survival Mechanics

### Who Survives Regime Change

| Factor | Survival Boost |
|--------|----------------|
| **Relationship with new regime** | +30 to +50 |
| **High drawing power** | +20 to +40 |
| **Irreplaceable skill** | +15 to +25 |
| **Long contract** | +20 (can't easily fire) |
| **Low salary** | +10 (cost-effective) |
| **Political flexibility** | +10 to +20 |

### Who Doesn't Survive

| Factor | Survival Penalty |
|--------|------------------|
| **Close to old regime** | -30 to -50 |
| **High salary, low draw** | -20 to -30 |
| **Political enemy of new regime** | -30 to -40 |
| **Style misfit** | -15 to -25 |
| **Backstage reputation issues** | -10 to -20 |

## Regime Change Phases

### Phase 1: Announcement (Week 1)

| Element | Description |
|---------|-------------|
| **Uncertainty spikes** | Roster morale drops 10-20% |
| **Rumor mill** | Speculation about changes |
| **Political maneuvering** | Workers position themselves |
| **External interest** | Competitors scout vulnerable talent |

### Phase 2: Assessment (Weeks 2-4)

| Element | Description |
|---------|-------------|
| **Meetings** | New regime evaluates roster |
| **Loyalty tests** | Subtle and explicit |
| **Early signals** | First booking decisions reveal philosophy |
| **Alliance formation** | New cliques form |

### Phase 3: Restructure (Months 1-3)

| Element | Description |
|---------|-------------|
| **Departures begin** | First wave of exits |
| **Push changes** | New favorites emerge |
| **Style shifts** | Product direction clarifies |
| **Resistance crystallizes** | Those who won't adapt identified |

### Phase 4: New Normal (Months 3-6)

| Element | Description |
|---------|-------------|
| **Stability returns** | Uncertainty diminishes |
| **New hierarchy** | Power structure settled |
| **Adaptation complete** | Survivors have adjusted |
| **Grudges persist** | Long-term political scars |
