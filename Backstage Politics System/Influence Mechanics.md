## Overview

Influence represents a worker's backstage political power: their ability to affect booking decisions, protect their position, and sway management. High-influence workers can demand creative control, bury rivals, and survive regime changes. Low-influence workers are vulnerable to political maneuvering.

## Influence Calculation (Conceptual)

```
Total Influence = Position Base + Tenure Modifier + Relationship Bonus + Personality Modifier + Recent Events
```

**Example Calculation:**

Worker: 8-year veteran, current midcarder, close to booker, Backstage Politician archetype

- Position (Midcard): +5
- Tenure (8 years): +15
- Relationship (Close to booker): +20
- Personality (Politician): +15
- **Total: 55 Influence**

## Influence Effects

### High Influence (60+)

| Ability | Description |
|---------|-------------|
| **Creative Input** | Can veto storylines they dislike |
| **Protected Position** | Difficult to demote or fire |
| **Talent Acquisition** | Can request specific hires/firings |
| **Booking Control** | Can demand finishes, opponents |
| **Information Access** | Knows plans before others |

### Moderate Influence (30-59)

| Ability | Description |
|---------|-------------|
| **Voice in Room** | Opinions heard but not decisive |
| **Some Protection** | Not easily buried |
| **Negotiation Leverage** | Better contract terms |
| **Clique Membership** | Can join powerful alliances |

### Low Influence (0-29)

| Ability | Description |
|---------|-------------|
| **Limited Voice** | Requests ignored |
| **Vulnerable** | Can be buried or released |
| **Weak Negotiation** | Take what's offered |
| **Political Target** | May be scapegoated |

### Negative Influence (-1 and below)

| Consequence | Description |
|-------------|-------------|
| **Active Burial** | Others work against them |
| **Blacklist Risk** | May be frozen out |
| **No Allies** | Isolated, vulnerable |
| **Exit Pressure** | May be forced out |
