# SCHEME DISCOVERY & INFORMANT NETWORK SYSTEM

üìõ **NAME:** Scheme Discovery & Informant Network  
üß≠ **CATEGORY:** Backstage Politics, Worker Agency, Locker Room Dynamics  
üîë **KEYWORDS:** snitching, informants, road agents, locker room standing, scheme exposure, backstage politics, worker relationships, intervention, blackmail, leverage, dirtsheets, whistleblowing  
üìù **SUMMARY:**

When workers scheme (refuse jobs, politic for spots, negotiate with rivals, sabotage peers), they don't operate in a vacuum. Other workers, road agents, and locker room informants may discover the scheme before it succeeds. Whether they REPORT it depends on their own relationships, motivations, personality, locker room atmosphere, and assessment of the schemer's value. This creates a counterbalance to the Shoot Betrayal system and makes locker room management meaningful beyond morale.

‚öôÔ∏è **LOGIC OVERVIEW:**

- Schemes have a discovery window before resolution
- Discovery chance based on: scheme type visibility, schemer personality, emotional state, time duration, witness pool size
- Weekly discovery rolls compound over time (small chances snowball)
- Informants are NOT passive snitch-bots; they make decisions based on their own personality and relationships
- Informant decision tree: Snitch (to booker or dirtsheets) / Intervene / Ignore / Co-Sign / Leverage
- Locker room atmosphere modifies all informant behavior (harmonious = internal handling, fractured = leaking normalized)
- Intervention adds a "talk them down" phase where respected veterans can stop schemes
- Player receives warnings if scheme is exposed to them, enabling counter-play
- If NOT exposed, scheme resolves and player deals with consequences
- Criminal schemes and whistleblowing operate on separate tracks with different moral weight

üî¨ **LLM INTEGRATION:** Yes, HIGH. LLM can generate:
- Informant reasoning ("I like the kid but he's gonna burn himself")
- Intervention dialogue (veteran talks down hothead)
- Snitch reports to booker with personality flavor
- Schemer response to intervention attempts
- Blackmail/leverage confrontations
- Whistleblower dilemmas

üìå **ORIGIN:** Developed in Claude conversation (Dec 2024), building on Vol 4 backstage politics concepts. Calibrated through historical examples (Montreal Screwjob, Hogan/WCW politicking, Plane Ride From Hell, AEW Brawl Out, Ring Boy Scandal, Vince allegations). Not extracted from original volumes but consistent with Hidden Personality and Agent AI design philosophy.

üìé **CONNECTED SYSTEMS:**
- **HIDE-001: Hidden Personality** - Schemer's Ego/Loyalty/Paranoia drives scheme initiation and detectability
- **SCHEME-001: Shoot Betrayal System** - Parent system; this is the COUNTER-system
- **REL-001: Relationship System** - Informant-to-schemer and informant-to-booker relationships determine response
- **MORALE-001: Morale System** - Booker's locker room standing affects discovery network; locker room atmosphere is key modifier
- **AGENT-001: Road Agent System** - Agents as primary intelligence source (embedded vs stooge distinction)
- **POLITICS-001: Backstage Politics** - Broader power struggle context
- **CATA-001: Catastrophic Events** - Criminal schemes escalate to this system
- **WHIST-001: Whistleblowing** - Crimes with victims trigger separate moral track

## FORMULA 3: WEEKLY DISCOVERY ROLL

For each person in pool, do they notice this week?

```
Weekly_Discovery_Chance = Base_Rate √ó Detectability √ó Witness_Awareness + Agent_Bonus
```

### Base_Rate

**0.05 to 0.10 (5-10%)** per week, all things average.

Small but compounds over time. Rewards quick decisive schemes, punishes drawn-out politicking.

### Witness_Awareness

| Type | Multiplier | Rationale |
|------|------------|-----------|
| Actively suspicious of schemer | √ó2.0 | Looking for it |
| Generally observant (road agent, vet) | √ó1.5 | Seen it all before, notices patterns |
| Average | √ó1.0 | Baseline |
| Self-absorbed (high ego, own problems) | √ó0.7 | Not paying attention to others |
| Oblivious (low awareness, green) | √ó0.5 | Doesn't know what to look for |

### Agent_Bonus (flat addition if witness is road agent)

| Agent Type | Flat Bonus | Rationale |
|------------|------------|-----------|
| Embedded with the boys | +0.10 to +0.15 | Trusted, boys talk freely around them |
| Management stooge | +0.0 | Boys clam up, hide things |
| Despised stooge | -0.05 | Boys actively work around them, extra careful |

### Examples

**Average witness, average scheme, average week:**
```
0.075 √ó 0.5 √ó 1.0 = 3.75% chance this person notices this week
```

**Observant veteran watching a loudmouth sandbag:**
```
0.075 √ó 1.0 (capped detectability) √ó 1.5 = 11.25% per week
```

**Embedded road agent watching same loudmouth:**
```
0.075 √ó 1.0 √ó 1.5 + 0.125 = 23.75% per week
```

## FORMULA 4: CUMULATIVE DISCOVERY OVER TIME

```
Per_Witness_Cumulative = 1 - (1 - Weekly_Chance)^Weeks

Any_Discovery = 1 - ‚àè(1 - Individual_Cumulative) for all witnesses
```

Small weekly chances snowball. Larger pools multiply opportunities.

### Examples

**3 witnesses, 5% weekly chance, 8 weeks:**
```
Per witness: 1 - (0.95)^8 = 33.7%
Any discovery: 1 - (0.663)^3 = 70.8%
```

**10 witnesses, 10% weekly chance, 4 weeks:**
```
Per witness: 1 - (0.90)^4 = 34.4%
Any discovery: 1 - (0.656)^10 = 98.3%
```

### Design Implication

- Instant schemes (Montreal): low discovery
- Month+ schemes (Hogan politicking): near guaranteed discovery
- Scheme duration is a meaningful tactical choice

## LEVERAGE MECHANICS

### Leverage Secrecy Status

| Status | Leverage Value |
|--------|----------------|
| Secret (only informant knows) | Full leverage |
| Rumored (dirtsheets suspect, unconfirmed) | Partial leverage (confirmation still has value) |
| Public (everyone knows) | Zero leverage |

**Critical Rule:** Leverage requires secrecy. Once public, leverage evaporates regardless of severity.

### Leverage Expiration (If Unused)

| Leverage Severity | Expiration |
|-------------------|------------|
| Minor (small rule break, being late) | 4-6 weeks |
| Moderate (refused job, mild politicking) | 3-6 months |
| Severe (active sabotage, secret negotiations) | 1-2 years |
| Nuclear (career-ending if exposed) | Indefinite |
| Criminal | Indefinite (or statute of limitations) |

### Leverage Usage Limits

| Severity | Uses Allowed |
|----------|--------------|
| Minor | One favor, debt cleared |
| Moderate | 2-3 favors over time |
| Severe | Ongoing leverage until relationship breaks |
| Nuclear/Criminal | Lifetime leash (while secret) |

### Favor Types

| Category | Examples |
|----------|----------|
| **Career** | Job to me, support my push, put me over |
| **Political** | Back me in disputes, vote my way, endorse me |
| **Scheme assistance** | Help my scheme, sabotage my enemy |
| **Protection** | Stay out of my business, don't work against me |
| **Humiliation** (enemies) | Public apology, embarrassing stipulation match |

### Refusal Consequences

When schemer refuses leverage demand, informant can:

| Response | Trigger |
|----------|---------|
| Snitch immediately | Bombthrower, low patience |
| Escalate demands | Backstage Politician, calculated |
| Wait and try again | Patient, strategic |
| Expose with amplification ("and they refused to make it right") | Vindictive, maximum damage |

## WHISTLEBLOWING TRACK

When informant discovers criminal behavior WITH A VICTIM, the system shifts from snitching/leverage to whistleblowing.

### Trigger Condition

```
IF Scheme_Type = Criminal
  AND Victim_Exists = True
  ‚Üí Whistleblowing track (separate from leverage calculus)
```

### Whistleblower Dilemma

| Approach | Evidence Weight | Career Risk | Believability |
|----------|-----------------|-------------|---------------|
| Anonymous tip to police | Low | Low | "Unverified source" |
| Anonymous tip to media | Low-Medium | Low | "Anonymous allegations" |
| Named report to police | High | High | Taken seriously |
| Named report to media | Very High | Very High | "Accuser goes public" |
| Lawsuit | Maximum | Maximum | Legal discovery |

### Power Differential Effect

| Accused Power Level | Believability Modifier |
|---------------------|------------------------|
| Jobber/low card | Believed easily |
| Midcard | Moderate scrutiny |
| Main eventer | "He said/she said" |
| Locker room leader | Industry closes ranks |
| Management | Active suppression |
| Owner | Near impossible without overwhelming evidence |

### Corroboration Mechanics

```
Accusation_Weight = Base_Evidence √ó (1 + (Additional_Accusers √ó 0.5)) √ó Era_Modifier √ó (1 / Power_Level)
```

One accuser vs monster = usually loses.
Multiple accusers = tide turns.

### Whistleblower Decision Factors (AI)

```
Factors increasing likelihood:
  + High morality
  + Close relationship to victim
  + Secure career position
  + Post-#MeToo era
  + Others also know (safety in numbers)
  + Community-Minded personality

Factors decreasing likelihood:
  + Fear of retaliation
  + Financial dependence on accused/promotion
  + NDA/legal threats
  + High Traditionalism ("don't air dirty laundry")
  + Victim asked them not to
  + Benefits from accused's success
```
