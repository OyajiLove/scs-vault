## Overview

Cliques are informal backstage alliances that form organically based on personality compatibility, shared interests, and mutual benefit. Unlike on-screen factions, cliques operate behind the curtain, influencing booking decisions, protecting members, and sometimes destabilizing promotions.

---

## LOCKER ROOM FACTION FORMATION (#111)

üìå **Source:** Vol 1 Extraction #111

**Core Concept:** Just like stables in the ring, backstage factions form organically based on relationships.

### Relationship Types Detected

| Type | Description | Detection Method |
|------|-------------|------------------|
| **Family** | Blood, marriage | Known data |
| **Friends/Allies** | Based on factions, backstage cliques, ride partners | Behavior tracking |
| **Romantic** | If public info available | Media, known relationships |
| **Rivalry/Hatred** | Shoot references, backstage incidents | Event tracking |
| **Loyalty** | Trainer, mentor, promoter, dojo loyalty | Training history |

### Relationship Strength Scale

| Level | Description |
|-------|-------------|
| **Shallow** | Acquaintances, polite but not invested |
| **Moderate** | Friends, will support but not at great cost |
| **Deep** | True allies, will sacrifice for each other |

### Event-Driven Updates

Relationships evolve through gameplay events:

| Event | Relationship Effect |
|-------|--------------------|
| Betrayal during contracts | -50 to -80 relationship |
| Helping injured friend | +20 to +40 relationship |
| Booking ally poorly | -10 to -30 relationship |
| Booking enemy poorly | +10 to +20 from allies |
| Romantic collapse | Scandal trigger, -100 relationship |

**Triangle Example:** Edge/Lita/Matt Hardy situation creates:
- Edge-Lita: Deep (romantic), then scandal
- Matt-Edge: Deep (friendship), then -100 (betrayal)
- Matt-Lita: Deep (romantic), then -100 (betrayal)
- Locker room split: Some side with Matt, others neutral

## CLIQUE POWER SYSTEM (#123)

üìå **Source:** Vol 1 Extraction #123

**Core Concept:** Veteran groups who quietly (or loudly) control booking, bury outsiders, or protect their own.

### Clique Power Actions

| Action | Effect |
|--------|--------|
| **Vote for/against pushes** | Influence booking committee decisions |
| **Poison the ear of booker** | Reduce target's push priority |
| **Protect members from burial** | Clique members harder to demote |
| **Target outsiders for humiliation** | Political burial of non-members |

### Backstage Political Tags

| Tag | Effect |
|-----|--------|
| **Locker Room Poison** | Causes morale drain, creates factions |
| **Locker Room Glue** | Helps resolve tension, keeps morale up |
| **Shit-Stirrer** | Plants rumors, fuels resentment |
| **Backstage Bully** | Intimidates, hazes lower card/rookies |

### Clique Member Effects

| Effect | Description |
|--------|-------------|
| **Protection** | Clique shields member from being buried |
| **Burial Power** | Can bury others politically |
| **Contract Leverage** | Clique support influences negotiations |
| **Information Network** | Early warning on booking plans |
| **Exit Difficulty** | Leaving clique creates enemies |

**Locked Example:** "A tag team split poisoned a locker room"
- Team breakup creates faction split
- Workers choose sides
- Morale across roster affected
- Booking becomes political minefield

## Clique vs. Faction Distinction

| Aspect | Clique (Backstage) | Faction (On-Screen) |
|--------|-------------------|---------------------|
| **Visibility** | Hidden from fans | Part of storyline |
| **Purpose** | Political power | Entertainment/booking |
| **Formation** | Organic | Booked |
| **Dissolution** | Personality conflicts | Storyline decisions |
| **Membership** | Invitation-based | Booked by creative |
| **Example** | The Kliq (1995) | nWo (1996) |

**Note:** Cliques may become factions (Kliq ‚Üí nWo) and factions may develop into cliques (spending time together creates genuine bonds).

## Clique Formation Triggers

### Compatibility Factors

| Factor | Compatibility Boost |
|--------|---------------------|
| **Similar archetypes** | High (Ace Complex attracts Ace Complex) |
| **Complementary archetypes** | Medium (Ace Complex + Loyal Soldier) |
| **Shared training background** | High (same dojo, same trainer) |
| **Similar career phase** | Medium (young workers bond, veterans bond) |
| **Shared grievances** | Very High (mutual enemy unites) |
| **Travel partners** | Medium (time together builds bonds) |
| **Similar styles** | Low-Medium (common ground) |

### Formation Algorithm (Conceptual)

```
Clique Probability = 
  (Personality Compatibility √ó 0.3) +
  (Shared Background √ó 0.2) +
  (Mutual Interest √ó 0.3) +
  (Opportunity √ó 0.2)
```

**Opportunity Factors:**
- Extended road trips together
- Same programs/feuds
- Shared locker room (tag teams, stables)
- Both new to promotion at same time

## Clique Types

### The Power Bloc

**Composition:** 2-4 high-influence workers
**Purpose:** Protect positions, control booking
**Example:** Hogan/Nash/Hall (WCW)

| Characteristic | Description |
|----------------|-------------|
| **Influence** | Very High combined |
| **Danger** | Can hold promotion hostage |
| **Weakness** | Ego clashes within |
| **Typical End** | One member gets more power, others resent |

### The Shield

**Composition:** Veterans protecting each other
**Purpose:** Mutual defense, career longevity
**Example:** Undertaker's locker room leadership

| Characteristic | Description |
|----------------|-------------|
| **Influence** | Moderate to High |
| **Danger** | Low (usually stabilizing) |
| **Weakness** | May resist new talent |
| **Typical End** | Retirements, natural dissolution |

### The Young Lions

**Composition:** Rising stars bonding
**Purpose:** Mutual support, shared ambition
**Example:** OVW Class (Cena, Orton, Batista, Lesnar)

| Characteristic | Description |
|----------------|-------------|
| **Influence** | Low initially, grows |
| **Danger** | May challenge veterans |
| **Weakness** | Competition as they rise |
| **Typical End** | Success breeds jealousy, splits |

### The Outcasts

**Composition:** Workers marginalized by mainstream
**Purpose:** Survival, alternative power base
**Example:** ECW originals in WWE

| Characteristic | Description |
|----------------|-------------|
| **Influence** | Low but united |
| **Danger** | May become exodus seed |
| **Weakness** | Limited political reach |
| **Typical End** | Absorbed or depart together |

## Clique Mechanics

### Membership Benefits

| Benefit | Description |
|---------|-------------|
| **Protection** | Members cover for each other |
| **Information** | Shared intel on booking plans |
| **Collective Leverage** | Negotiate as bloc |
| **Political Cover** | Blame deflection |
| **Career Boost** | Members promote each other |

### Membership Costs

| Cost | Description |
|------|-------------|
| **Loyalty Obligations** | Must support other members |
| **Enemy Inheritance** | Clique enemies become yours |
| **Independence Loss** | Can't make solo deals |
| **Target Status** | Cliques draw opposition |

### Clique Internal Dynamics

| Dynamic | Effect |
|---------|--------|
| **Leadership Struggle** | High-Ego members compete for top spot |
| **Loyalty Tests** | Members tested when interests conflict |
| **Information Hierarchy** | Not all members equally informed |
| **Exit Difficulty** | Leaving creates enemies |

## Clique Dissolution

### Dissolution Triggers

| Trigger | Probability |
|---------|-------------|
| **Leader departure** | 80% dissolution |
| **Member success beyond group** | 50% strain |
| **External pressure (management)** | 40% weaken |
| **Internal betrayal** | 90% dissolution |
| **Philosophical split** | 60% dissolution |
| **Time (5+ years)** | Natural drift |

### Post-Dissolution Outcomes

| Outcome | Description |
|---------|-------------|
| **Clean Break** | Members remain cordial |
| **Bitter Split** | Creates long-term feud |
| **Reformation** | Clique rebuilds under new terms |
| **Absorption** | Members join other cliques |
