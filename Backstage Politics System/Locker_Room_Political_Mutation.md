## 1. WHEN LOCKER ROOM MUTATIONS BEGIN

| Trigger | Description |
|---------|-------------|
| **Fanbase Subculture Fracturing** | Different fan groups start backing different styles, workers, divisions |
| **Worker Emotional Alignment** | Workers feel the pull: some naturally align with new emotional fan waves, others cling to old prestige |
| **Booking Misreads or Partial Panderings** | Management starts favouring certain emotional waves over others, intentionally or accidentally |
| **Clique Recruitment Starts** | Workers form political alliances based on style loyalty, fanbase alignment, shared sense of injustice or hope |
| **Golden Era Fragmentation** | As emotional unity fades, so does locker room unity |

### Trigger Thresholds

| Trigger | Threshold |
|---------|-----------|
| **Fan Subculture Variance** | 3+ major subcultures with 15%+ share each |
| **Booking Favoritism** | Noticeable resource imbalance toward one style |
| **Golden Era Duration** | 4+ years increases fragmentation risk |
| **Recent Scandal/Betrayal** | Accelerates clique formation |

## 2. CLIQUE TYPES THAT FORM

| Clique Type | Description | Fan Archetype Mirror |
|-------------|-------------|---------------------|
| **Purist Defenders** | Technical masters, psychology obsessives, old-school respect soldiers | Purists |
| **Crash TV Opportunists** | Chaos merchants, angle chasers, workers craving screen time over arc depth | Crash TV Fans |
| **Cynic Enablers** | Workers embracing irony, knowing the game, surviving by mocking sincerity | Cynics/Ironists |
| **Workrate Crusaders** | Five-star hunters, indie purists, relentless athletic elevationists | Workrate Fetishists |
| **Heat Brood** | Workers obsessed with heat, domination, heel runs. Often ruthless backstage players. | Heat Chasers |
| **Loyalist Isolationists** | Workers who defend promotion's brand over any booking philosophy. Cultists willing to go down with ship. | Cult Loyalists |

### Clique Characteristics

| Clique | Strengths | Weaknesses |
|--------|-----------|------------|
| **Purist Defenders** | Respected by peers, media darlings | Politically outmaneuvered by flashier groups |
| **Crash TV Opportunists** | Ratings spikes, management favor | Burn out fast, create chaos |
| **Cynic Enablers** | Survive any regime change | Never trusted fully |
| **Workrate Crusaders** | Critical acclaim, internet support | Struggle with casual audience |
| **Heat Brood** | Booking power through heat generation | Scapegoated when trust collapses |
| **Loyalist Isolationists** | Job security through loyalty | Seen as management stooges |

## 3. HISTORICAL EXAMPLES

| Example | Parallel |
|---------|----------|
| **WWF Kliq (1995-1996)** | HBK, Nash, Hall: Crash TV opportunists and political gatekeepers |

| **AEW Post-Brawl Out Era** | Punk loyalists vs Elite loyalists vs newer emotional wave supporters (Hangman, Swerve factions) |

Locker room splits mirror fan splits. Booking chaos threatens from inside the walls.

## 4. HOW POLITICAL MUTATIONS AFFECT PROMOTIONS

| Behavior | Result |
|----------|--------|
| **Booking Sabotage Risk** | Workers half-ass matches, sandbagging, politicized finishes. Cliques sabotage each other on TV. |
| **Factions Recruit Workers** | Cliques actively seek neutral midcarders, rookies, agents. Building political armies backstage. |
| **Contract Negotiation Tensions** | Workers from dominant cliques get better offers; exiled ones squeezed out or humiliated in renewals |
| **Booking Instability** | Title scenes, division pushes, main event booking become volatile. Loyalty/politics override pure storytelling. |
| **Fanbase Perception Shifts** | Hardcore fans notice the fractures. Crowd reactions mutate depending on which cliques they subconsciously align with. |

### Sabotage Types

| Sabotage | Effect |
|----------|--------|
| **Half-Assed Matches** | Match quality drops, crowd notices |
| **Sandbagging** | Making opponent look bad through non-cooperation |
| **Politicized Finishes** | Refusing to job clean, demanding finish changes |
| **Shoot Promos** | Real grievances aired on live TV |
| **No-Sell Campaigns** | Refusing to sell opponent's offense properly |

## 5. LONG-TERM POLITICAL FALLOUT

| If Clique Wars Escalate... | Simulation Behavior |
|---------------------------|---------------------|
| **Promotion Split or Collapse** | Mass walkouts, coup attempts, shoot promos, external wars (TNA formation, AEW fractures) |
| **New Promotion Formations** | Exiled cliques form new promotions with "purified" emotional philosophies (Honor Club-era ROH purist refugee wave) |
| **Fan Emotional Draining** | As backstage chaos leaks into booking, fans grow disillusioned faster, even in technically good matches |
| **Historical Memory Burned** | Promising Golden Eras remembered as tragic lost potential stories ("What could have been if politics hadn't killed it") |

### Escalation Ladder

| Level | State | Consequences |
|-------|-------|--------------|
| **1: Tension** | Cliques forming, minor friction | Booking slightly affected |
| **2: Cold War** | Open hostility, recruitment campaigns | Match quality suffers |
| **3: Active Conflict** | Sabotage attempts, shoot promos | Fans notice, trust erodes |
| **4: Civil War** | Open warfare, management paralyzed | Promotion in crisis |
| **5: Collapse/Exodus** | Mass departures, new promotions form | Promotion may not survive |
