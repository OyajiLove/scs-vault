## Core Concept

In real wrestling, referees are not just counting pins. They are critical emotional controllers, audible deliverers, and match story enforcers.

In SCS, referees will:
- Deliver mid-match audibles (signals, whispers, earpieces depending on era/promotion)
- Influence match outcomes subtly
- Risk making mistakes (bad communication, botched messages)
- Sometimes even betray (if loyalty factors trigger, very rare but huge when it happens)

## Three Communication Methods âœ… LOCKED

| Method | Description | Era/Promotion Examples |
|--------|-------------|----------------------|
| **Signal-Based** | Visual gestures, whispered messages. Old-school style. | 70s-90s US, Japan, early ECW, older lucha libre |
| **Earpiece Live** | Ref wired to back. Agents or bookers feed changes in real time. | Modern WWE, AEW, NJPW Dome shows |
| **Hybrid** | Combination of hand signals + live mics. Flexible per worker or situation. | Some modern indies, AEW middle card |

Promotions have internal settings for which systems they primarily use. Some shows (big TV, PPVs) may force earpiece use even in otherwise "signal-based" promotions.

## Example: Referee Audible in Action

**Situation:** Title match planned for 30 mins. Crowd visibly fading by minute 12. Agent signals ref via earpiece.

**Referee Action:** Ref bumps into Worker A during tie-up. Quietly mutters: "Cut to nearfall sequence. Go home in 3."

**Worker Response:**
- If high Psych/Adaptability/Focus: Adjusts smoothly.
- If low stats: Botches nearfall, crowd notices awkwardness.

**World Memory Impact:**
- If saved well: Match remembered as brave gritty survival.
- If botched: Loyalty drift against promotion/booking exposed.
