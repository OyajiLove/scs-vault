## Overview

Territorial Loyalty represents a worker's attachment to their home region or adopted territory. High territorial loyalty workers resist leaving their turf, wield influence locally, and may become "Territorial Kingpins" who shape regional booking. Low loyalty workers travel freely but never become local icons.

---

## Territorial Loyalty Scale

| Level | Range | Descriptor | Behavior |
|-------|-------|------------|----------|
| **Extreme** | 18-20 | "Lifer" | Will never leave, demands booking control |
| **High** | 14-17 | "Hometown Hero" | Prefers home, needs strong incentive to leave |
| **Moderate** | 10-13 | "Regional Star" | Home base preference, will tour |
| **Low** | 6-9 | "Journeyman" | No strong attachment, follows money |
| **None** | 1-5 | "Drifter" | No home base, always moving |

## Territorial Kingpin Archetype

From [[_Hidden Personality System Index|Hidden Personality System]]:

**Territorial Kingpin:**
- **Core Concept:** Local legend who resents outsiders and modern styles
- **Stat Ranges:** Ego 14-20, Loyalty (to territory) 16-20, Traditionalism 15-20, Risk-Taking 2-7
- **Behavior:** Demands protection in hometown, morale drops when foreigners get top billing, can block pushes
- **Inspiration:** Jerry Lawler (Memphis), Wahoo McDaniel, Nick Gulas era
- **Tags:** "Territory Lifer", "Disgruntled Veteran", "Locker Room Politician"

## Loyalty Effects

### High Territorial Loyalty (14+)

| Effect | Description |
|--------|-------------|
| **Booking Influence** | Can veto outsider pushes in home territory |
| **Crowd Connection** | +20% pop in home territory |
| **Contract Leverage** | Demands territory-specific clauses |
| **Morale Sensitivity** | Drops if outsiders pushed over them at home |
| **Departure Resistance** | Very hard to poach |

### Moderate Territorial Loyalty (10-13)

| Effect | Description |
|--------|-------------|
| **Home Preference** | Will tour but returns home |
| **Crowd Connection** | +10% pop in home territory |
| **Contract Flexibility** | Accepts touring clauses |
| **Morale Stability** | Tolerates outsider pushes |
| **Departure Possibility** | Can be poached with right offer |

### Low Territorial Loyalty (1-9)

| Effect | Description |
|--------|-------------|
| **No Home Base** | Moves freely |
| **Crowd Connection** | No regional bonus |
| **Contract Flexibility** | Takes best offer anywhere |
| **Morale Stability** | Not affected by regional politics |
| **Departure Ease** | Easy to poach |

## Loyalty Formation

### Factors Increasing Territorial Loyalty

| Factor | Loyalty Increase |
|--------|------------------|
| **Born in territory** | +3 base |
| **Trained in territory** | +2 base |
| **Family in territory** | +2 base |
| **Title reign in territory** | +1 per reign |
| **5+ years in territory** | +1 per additional year |
| **Crowd love** | +1 per high-pop milestone |
| **Territorial Kingpin mentorship** | +2 |

### Factors Decreasing Territorial Loyalty

| Factor | Loyalty Decrease |
|--------|------------------|
| **Buried in home territory** | -2 |
| **Passed over for outsider** | -1 per occurrence |
| **Territory declining** | -1 per year of decline |
| **Better offer elsewhere** | -1 to -3 (temptation) |
| **Personal conflict with booker** | -2 |
| **Family moved away** | -2 |

## Territorial Loyalty Conflicts

### Kingpin vs. Outsider

When a Territorial Kingpin faces an incoming star:

| Scenario | Kingpin Response |
|----------|------------------|
| **Outsider pushed above Kingpin** | Morale crash, political resistance |
| **Outsider as partner** | Grudging acceptance if Kingpin dominates |
| **Outsider defeats Kingpin** | Demands rematch clause, may sabotage |
| **Outsider respects Kingpin** | Possible mentorship, gradual acceptance |

### Multiple Kingpins

Some territories have multiple Kingpins with overlapping influence:

| Relationship | Effect |
|--------------|--------|
| **Allied Kingpins** | Territory extremely stable, hard to break in |
| **Rival Kingpins** | Territory politically volatile, opportunities exist |
| **Generational Kingpins** | Old guard vs. new guard tension |
