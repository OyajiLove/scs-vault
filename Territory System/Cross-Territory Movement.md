## Overview

Cross-Territory Movement governs how workers travel between territories, whether through official talent exchanges, touring agreements, invasions, or defections. These movements create storyline opportunities, culture clashes, and economic tensions.

---

## Movement Types

### Official Exchange

**Definition:** Territories agree to swap talent temporarily
**Duration:** Typically 2-8 weeks
**Relationship Required:** Friendly or neutral territory relations

| Element | Description |
|---------|-------------|
| **Purpose** | Fresh matchups, worker development |
| **Terms** | Split gate, shared promotion |
| **Reception** | Announced as visiting champion/star |
| **Return** | Worker returns to home territory |

### Touring

**Definition:** Worker appears in multiple territories as traveling attraction
**Duration:** Continuous or periodic
**Relationship Required:** NWA-style agreement or GWA national deal

| Element | Description |
|---------|-------------|
| **Purpose** | Draw money across territories |
| **Terms** | Worker contracted nationally, tours regionally |
| **Reception** | Treated as special attraction |
| **Example** | NWA World Champion touring territories |

### Invasion

**Definition:** One territory sends talent to "attack" another
**Duration:** Until storyline resolution
**Relationship Required:** Hostile or competitive

| Element | Description |
|---------|-------------|
| **Purpose** | Angle fuel, interpromotional feud |
| **Terms** | One territory's talent "invades" another |
| **Reception** | Automatic heat, outsider threat |
| **Resolution** | War games, champion vs. champion, surrender |

### Defection

**Definition:** Worker leaves one territory for another
**Duration:** Permanent
**Relationship Required:** N/A (often damages relations)

| Element | Description |
|---------|-------------|
| **Purpose** | Better deal, creative freedom, escape |
| **Terms** | Contract buyout, expiration, or breach |
| **Reception** | Complex (traitor to old, unknown to new) |
| **Consequences** | May trigger blacklisting, legal action |

### Poaching

**Definition:** Territory actively recruits workers from competitors
**Duration:** N/A (recruitment action)
**Relationship Required:** Hostile or indifferent

| Element | Description |
|---------|-------------|
| **Purpose** | Weaken competitor, strengthen roster |
| **Terms** | Better contract offer |
| **Reception** | Territory war escalation |
| **Consequences** | Retaliatory poaching, industry tensions |

## Movement Mechanics

### Exchange Agreements

```
Exchange Probability = (Territory Relationship √ó 0.4) + (Worker Value √ó 0.3) + (Mutual Benefit √ó 0.3)
```

| Territory Relationship | Base Probability |
|------------------------|------------------|
| **Allied** | 80% |
| **Friendly** | 60% |
| **Neutral** | 40% |
| **Tense** | 15% |
| **Hostile** | 5% |

### Invasion Angle Setup

**Requirements:**
1. Two territories with tense or hostile relationship
2. Worker(s) willing to appear as invaders
3. Storyline justification (title claim, personal feud, territory war)

**Phases:**
| Phase | Description |
|-------|-------------|
| **1. Provocation** | Comments, challenges from afar |
| **2. Incursion** | First appearance, run-in or attack |
| **3. Response** | Home territory retaliates |
| **4. Escalation** | Multi-man conflict, recruits on both sides |
| **5. Resolution** | Big match(es) to settle conflict |

### Defection Process

| Stage | Description |
|-------|-------------|
| **Contract Status** | Check if worker can legally leave |
| **Negotiation** | New territory makes offer |
| **Decision** | Worker weighs options (see below) |
| **Announcement** | Departure becomes public |
| **Reaction** | Old territory responds (blacklist, legal, etc.) |
| **Debut** | Worker appears in new territory |

### Defection Decision Factors
