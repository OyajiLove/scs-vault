## Promo Response Types

| Response Type | Description | Drift Outcome |
|---------------|-------------|---------------|
| Acknowledge Softly | Accepts fan reaction without pandering | Begins slow-burn face turn |
| Direct Reversal | Openly thanks fans, turns face in promo | Full turn locked in |
| Deflect/Mock | Calls fans fickle, untrustworthy | Extends ambiguity, crowd heat grows |
| Gaslight | Claims fans are still booing even as they cheer | Creates heat paradox, can hold heel status if committed |
| Weaponize Drift | Blames face workers for "stealing attention," fans for being "idiots" | Heel doubles down, nuclear heat segment |
| Split Crowd Lean | Uses dueling chants to cut promos tailored to both halves | Creates modern anti-hero dynamic (Cena, MJF) |

## Promo Tone Flags

| Tone Flag | Effect |
|-----------|--------|
| Vulnerable | Increases emotional resonance, Buzz spike likely |
| Cold / Detached | Delays drift, fans project mystery onto worker |
| Mocking | Generates polarizing crowd split |
| Gratitude | Auto-triggers Memory Snapshot if combined with farewell, title win, or save |
| Denial | Risky: if crowd doesn't buy it, segment flops or turns ironic |
| Breaking Character | "Real name" drop, shoot-style, may trigger Myth-Making if emotionally lands |

These tones shape how segment text appears in summaries and how fans remember it.
