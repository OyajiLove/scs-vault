## Swerve vs Hangman Case Study âœ… LOCKED

**Source:** Vol 5 lines 960-964

The Swerve Strickland vs Hangman Page feud (AEW, 2023-2024) is the gold standard for Living Feud Memory:

### Feud Evolution Stages

| Stage | Event |
|-------|-------|
| **Stage 1: Real Wound Created** | Swerve invades Hangman's privacy and family life: deep emotional violation anchored in real-world fan emotion |
| **Stage 2: Audience Investment** | Fans feel the betrayal. Not about title belts, about emotional violation of trust, safety, sanctity. |
| **Stage 3: Grudge Deepens** | Matches aren't about "who wins": they're about wounding, payback, survival of self-respect |
| **Stage 4: Legendary Blood Feud** | Even when Swerve moves to other arcs, the Hangman wound is never fully closed. Lingers organically. |
| **Stage 5: Memory Lingers** | Even without constant promos, crowd remembers. Future re-encounters carry heat naturally, no rebuild needed. |
| **Stage 6: Reluctant Partners** | Forced to team out of necessity against overwhelming threats (Death Riders). Emotional scars still present beneath forced alliance. |

### Lessons for SCS

| Lesson | Detail |
|--------|--------|
| **Living Emotional Anchor** | Feuds that violate real emotional safety (family, betrayal, humiliation) live on. Match results do not "reset" emotional scars. |
| **World Memory Gravity** | Crowd reactions evolve with or without booking. Future matches carry residual scar tissue whether intentionally booked or not. |
| **No "Win = Forgiveness"** | Even if Hangman beat Swerve decisively, the emotional scar would remain. Some wounds can't be "pinned away." |
| **Future Booking Influenced** | If Hangman becomes champion and Swerve challenges, feud memory reignites organically. No fresh build needed. |
| **Worker Emotional Damage** | Hangman emotionally "regressed" post-Swerve (drinking, depression tones returning). True feud memory shaped his arc beyond the matches. |

### How SCS Will Model This

| Layer | Behavior |
|-------|----------|
| **Emotional Feud Anchors** | Real betrayal arcs create permanent hidden feud memory anchors between workers |
| **Crowd Emotional Memory Drifts** | If workers re-encounter years later, fans emotionally surge without fresh rebuilds |
| **Worker Psychological Memory** | High Psychology/Focus workers "remember" betrayals in promos, selling, feud ignition even years later |
| **Booking Flexibility** | Players/AI can trigger callbacks or ignore them, but world memory still influences heat and reactions |

**Philosophy:** No mechanical feud timers. Only living emotional scar tissue.

## Core Concept

In SCS, feuds aren't just event-to-event storylines. They leave emotional scars on:
- Workers
- Fans
- Promotions
- World memory

Some feuds become legends. Some feuds become tragic regrets. Some feuds rot and die midstream, forgotten by all.

Feud memory shapes:
- Booking loyalty
- Crowd reactions years later
- Career arcs
- Promotion emotional myths

---

## How Feud Memory Forms âœ… LOCKED

| Type | Behavior |
|------|----------|
| **Legendary Blood Feuds** | Wars that escalate emotionally, climax naturally, scar fans and workers permanently (Misawa vs Kawada) |
| **Tragic Betrayal Feuds** | Worker betrayal arcs that never fully heal. Even post-feud, emotional drift lives (Bret vs HBK). |
| **Forgotten Feuds** | Feuds that lacked emotional spark or were cut off awkwardly rot into obscurity |
| **Cold War Feuds** | Quiet backstage or emotional resentments that flare over decades (e.g., workers refusing to work with certain people even in reunions) |

Real world emotional texture. Not mechanical "feud completed: +5 heat" garbage.
