## Overview

Not everyone remembers the same thing the same way. Fan types have distinct memory patterns.

---

## Fan Type Memory Styles

| Fan Type | Memory Style | What They Remember |
|----------|--------------|-------------------|
| **Hardcore Fans** | Deep recall, emotional weight, legacy-rich | Everything, with context |
| **Casual Fans** | Memory fades fast; follow trends, not arcs | Big moments only |
| **Cultural Loyalists** | Region-first memories | "Homegrown hero" bias |
| **Workrate Heads** | Remember bangers, forget promos | Match quality, move sequences |
| **Promo Junkies** | Remember segments, viral shoots | Pipebombs, catchphrases |
| **Trolls & Contrarians** | Misremember on purpose | Ironic chants, revisionism |

---

## Memory Persistence by Type

| Fan Type | Decay Speed | Distortion Risk |
|----------|-------------|-----------------|
| **Hardcore** | Slow | Low (but biased) |
| **Casual** | Fast | High |
| **Cultural Loyalist** | Medium (slow for locals) | Regional bias |
| **Workrate Head** | Medium | Low for matches, high for stories |
| **Promo Junkie** | Medium | Moderate |
| **Troll** | Variable | Intentional distortion |

---

## Fan Type Distribution

Varies by region and era:

| Region/Era | Dominant Types |
|------------|---------------|
| **1980s US** | Casuals, Cultural Loyalists |
| **1990s Japan** | Workrate Heads, Hardcore |
| **2000s Internet Era** | Hardcore, Trolls emerging |
| **2020s US/UK** | Trolls, Promo Junkies, split Hardcore |
| **Mexico (any era)** | Cultural Loyalists, Hardcore |

---

## Conflicting Memories

When fan types clash:

| Conflict | Example |
|----------|---------|
| **Hardcore vs Casual** | "That run was legendary" vs "Who?" |
| **Workrate vs Promo** | "Best worker ever" vs "Couldn't cut a promo" |
| **Loyalist vs Outsider** | "Our hero" vs "Regional nobody" |
| **Troll vs Everyone** | "Actually he was overrated" |
