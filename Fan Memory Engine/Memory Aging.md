## Overview

How memories fade and transform over time.

---

## Decay Stages

| Timeframe | State | Description |
|-----------|-------|-------------|
| **0-6 Months** | Fresh | Recalled clearly and referenced often |
| **6-24 Months** | Softening | Specifics fade, but vibe remains |
| **2-5 Years** | Distorted | Fans start to [[Memory Misremembering|misremember]]; narrative becomes stylized |
| **5+ Years** | Myth or Forgotten | Becomes symbolic (if strong) or lost entirely (if weak) |

---

## Decay Speed Modifiers

| Modifier | Effect on Decay |
|----------|-----------------|
| **Buzz at Time** | High = slower decay |
| **Emotional Tag** | Catharsis/Betrayal = slower decay |
| **Commentary Reinforcement** | Referenced often = slower decay |
| **Region of Occurrence** | Faster decay elsewhere; slower locally |
| **Anniversaries / Comebacks** | Reactivates (temporarily halts decay) |
| **Promotion Size** | Big moments in big feds linger longer |
| **Fan Type** | Casuals forget faster; hardcores preserve |
