## Overview

Modern crowds contaminate heel runs if not carefully managed. System tracks the line between Heat and Hate.

---

## Tools for Maintaining True Hate

| Tool | Function |
|------|----------|
| "No Merch" Clause | AI/booker refuses to release heel merch (resists cool-factor drift) |
| Scorned Segment Tone | Segments scripted as cruel, uncomfortable, not performative |
| Target Beloved Workers | Heel insults/attacks cultural icons (Immortals, Legends, Young Aces) |
| Cheap Heat Flags | City insults, kid bullying, off-limits mic work (crowd hates it, not enjoys it) |
| Stifled Finish Booking | Heels win via groan-worthy tactics, not cleverness |
| Isolation Protocol | Deny heel "tag in" to hot feuds, keep them in miserable heel vortex |
