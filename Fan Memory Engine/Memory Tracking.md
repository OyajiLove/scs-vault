## Overview

What the Fan Memory Engine tracks across the simulation.

---

## Memory Types

| Type | Description |
|------|-------------|
| **[[Memory Snapshots|Snapshot Moments]]** | Career-defining angles, matches, betrayals |
| **Feud Memory** | Heat and emotional resonance of long arcs |
| **Title Reigns** | Who defined the belt's prestige (not always based on wins) |
| **Promo Segments** | Shocking shoots, viral interviews, live meltdowns |
| **Worker Perception** | Myth vs reality; fanbase clings to early moments even after decline |

---

## Memory Layers

Each memory lives inside multiple layers:

| Layer | Function |
|-------|----------|
| **Region** | Where the memory is strongest |
| **Promotion** | Which company owns the memory |
| **Fan Type** | Which fans remember most vividly |
| **Time** | How old the memory is |

---

## Memory Strength

Tracked as 1-100 value per memory:

| Strength | Meaning |
|----------|---------|
| **80-100** | Vivid, frequently referenced |
| **50-79** | Active, shapes perception |
| **20-49** | Fading, needs reactivation |
| **1-19** | Nearly forgotten |

---

## Global vs Local Memory

| Scope | Behavior |
|-------|----------|
| **Global Memory** | Rare; only for legendary moments |
| **Regional Memory** | Most memories are region-locked |
| **Promotion Memory** | Company-specific recall |
