## Overview

Old memories can be reactivated, brought back to relevance. But often they return twisted.

---

## Reactivation Triggers

| Trigger | Effect |
|---------|--------|
| **Comebacks** | Instant reactivation of career memories |
| **Hall of Fame videos** | Curated memory refresh |
| **Anniversary shows** | Nostalgia spike for era |
| **AI booker nostalgia angles** | Deliberate memory leverage |
| **Fan campaigns** | "#RememberTheRun" style pushes |
| **Documentary release** | Major memory reframe |
| **Death** | Massive reactivation + mythologizing |

---

## Reactivation Effects

| Effect | Outcome |
|--------|---------|
| **Memory strength boost** | Faded memory returns to active |
| **Decay pause** | Temporarily halts further decay |
| **Potential distortion** | Memory may return twisted |
| **Legacy recalculation** | HOF consideration renewed |
| **Buzz spike** | If reactivation lands emotionally |

---

## Twisted Reactivation

When memories return, they often return changed:

| Original Memory | Reactivated Version |
|-----------------|---------------------|
| "Won the title in a good match" | "The greatest title win ever" |
| "Had a decent run" | "Criminally underrated" |
| "Popular heel" | "Misunderstood antihero" |
| "Lost in the main event" | "Should have won" |

---

## Reactivation Failure

Sometimes reactivation doesn't work:

| Cause | Result |
|-------|--------|
| **Too much time passed** | Fans don't care |
| **Replaced by new favorites** | Memory obsolete |
| **Scandal surfaced since** | Memory tainted |
| **Poor execution** | Comeback flops, memory damaged further |

---

## Intentional Reactivation

Players and AI bookers can deliberately trigger reactivation:

| Method | Risk/Reward |
|--------|-------------|
| **Video package** | Low risk, moderate reward |
| **Return match** | High risk, high reward |
| **Promo reference** | Low risk, low reward |
| **Anniversary angle** | Moderate risk, moderate reward |
