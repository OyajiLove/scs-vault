## Simulation Rules

### 1. Region + Era Gate Chant Pools
- Each combo has specific chant types available
- Crowd segments seeded with Fan Culture Profiles
- Engine uses Booking Memory + Emotional Tags + Worker Pop/Buzz to determine triggers

### 2. Tone Enforcement

| Tone | Description |
|------|-------------|
| Earnest | Loyal, direct, not mocking ("We love you!") |
| Tribal | Based on identity, face/heel allegiance ("USA!") |
| Ironic | Modern smark behavior, booing the good guy, chanting for nonsense |
| Ritualized | Puro/lucha/joshi stylized respect behaviors |
