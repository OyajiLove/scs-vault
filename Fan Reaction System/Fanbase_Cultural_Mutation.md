## 1. HOW FAN CULTURES FORM INITIALLY

| Formation Driver | Description |
|------------------|-------------|
| **Promotion Founding Identity** | Promotions imprint initial "spirit" onto fanbase: technical mastery (AJPW), defiant chaos (ECW), spectacle (WWF Golden Era) |
| **Early Booking Philosophy** | Early booking habits create emotional expectations: long arcs, technical feuds, car crash spotfests, heel heat dominance |
| **National/Regional Cultural Roots** | U.S. vs Mexico vs Japan vs Europe base styles deeply affect early fan values |

### Initial Identity Examples

| Promotion | Founding Identity |
|-----------|-------------------|
| **AJPW (70s-80s)** | Technical mastery, strong style, serious competition |
| **ECW** | Defiant chaos, anti-corporate rebellion, blood and violence |
| **WWF Golden Era** | Spectacle, larger-than-life characters, family entertainment |
| **NJPW** | Serious strong-style, legitimate sport presentation |
| **Lucha Libre** | High-flying spectacle, mask tradition, family legacy |

## 2. HOW FAN CULTURES MUTATE OVER TIME

| Mutation Cause | Effect |
|----------------|--------|
| **Booking Habits Shift** | If promotions switch from arc-based booking to crash TV, fans evolve into spectacle chasers or irony-poisoned crowds |
| **Repeated Emotional Betrayals** | Cynicism blooms; fans become less emotionally invested, more detached/sarcastic, harder to satisfy |
| **Generational Change** | Younger fans raised on different booking styles grow into adulthood with radically different expectations than legacy fans |
| **Superworker Golden Eras** | Belts, feuds, characters all deeply respected: creates a generation of "purists" valuing match integrity and emotional arcs |
| **Invasive External Influence** | Media landscape (social media discourse, streaming styles) accelerates cultural drift by blending international fandom norms |

### Mutation Speed

| Cause | Speed |
|-------|-------|
| **Booking Habits Shift** | Slow (1-3 years to notice) |
| **Repeated Betrayals** | Medium (6-18 months) |
| **Generational Change** | Very Slow (5-10 years) |
| **Golden Era** | Slow positive (2-5 years to solidify) |
| **External Cultural Shock** | Fast (months) |

## 3. CULTURAL ARCHETYPES THAT CAN EMERGE

| Fanbase Type | Characteristics |
|--------------|-----------------|
| **Purists** | Value long matches, psychology, clean finishes. Skeptical of crash TV. |
| **Crash TV Addicts** | Value car crashes, chaos, surprise moments over match quality |
| **Cynical Irony Fans** | Detached from emotional investment. Cheer swerve booking, mock sincerity. |
| **Bloodthirsty Spectacle Fans** | Worship brutality (deathmatches, hardcore, danger bumps) |
| **Loyalists** | Stick with promotion no matter what, even through Dark Eras |
| **Workrate Fetishists** | Value technical execution above emotional arcs. Matches as athletic exhibitions. |
| **Heat Merchants** | Love brutal, heel-dominant eras. Booing is joy. |
| **Cultists** | Treat promotion as religious devotion. Resent outsiders, crave ideological purity. |

### Archetype Booking Preferences

| Archetype | Loves | Hates |
|-----------|-------|-------|
| **Purists** | Long matches, clean finishes, storytelling | Overbooked finishes, interference, gimmick matches |
| **Crash TV Addicts** | Swerves, weapons, chaos | Slow builds, technical matches, "boring" psychology |
| **Cynical Irony Fans** | Meta-commentary, worked shoots, "smart" booking | Sincere emotional moments, traditional heroes |
| **Bloodthirsty Spectacle** | Deathmatches, danger bumps, blood | Safe wrestling, PG content |
| **Loyalists** | Anything their promotion does | Criticism, outsider perspectives |
| **Workrate Fetishists** | Technical skill, move variety, athletic exhibitions | Character work, promos, "sports entertainment" |
| **Heat Merchants** | Heel dominance, real heat, boos | Face-heavy booking, "feel good" moments |
| **Cultists** | Ideological purity, "true" wrestling | Mainstream appeal, crossover attempts |

## 5. GAMEPLAY IMPACT OF CULTURAL MUTATION

| Effect | Details |
|--------|---------|
| **Booking Success Windows Shift** | Booking styles that worked a decade ago might flop miserably if fanbase has mutated |
| **Promotions Can Misread Their Own Fans** | Hot-shotting "emotional arcs" to a crash TV crowd = failure. Booking serious title defenses to irony fans = dead air. |
| **Golden Eras Are Perishable** | Even purist, loyalist cultures eventually mutate into skeptical or bitter groups if not constantly emotionally fed |
| **Fanbase Segmentation** | Same promotion can end up with warring internal fan cultures that love and hate different booking styles, fueling future booking crises |

### Misread Warning Signs

| Symptom | Meaning |
|---------|---------|
| **Great matches getting silence** | Fanbase has mutated away from workrate appreciation |
| **Swerves getting groans instead of pops** | Cynical mutation complete, fans no longer buy surprises |
| **Blood getting walkouts** | Purist mutation, fans reject spectacle violence |
| **Sincere promos getting mockery** | Irony poisoning has set in |

## 6. FANBASE SEGMENTATION

When a promotion has been around long enough, it often contains multiple competing fan cultures:

| Segment | Behavior |
|---------|----------|
| **Legacy Fans** | Remember "the good old days," resistant to change |
| **New Generation** | Only know current product, different expectations |
| **Returning Lapsed Fans** | Expect the product they left, confused by changes |
| **Crossover Fans** | Came from another promotion, bring different values |

### Segmentation Conflict Examples

| Promotion | Conflict |
|-----------|----------|
| **AEW 2022+** | Elite loyalists vs Punk loyalists vs indie purists vs ex-WWE fans |
| **WWE 2015+** | Attitude Era nostalgists vs PG era kids grown up vs Network completists |
| **NJPW 2020s** | Japanese purists vs Western workrate tourists vs AEW crossovers |
