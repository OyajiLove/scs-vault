## 1. WHEN A GOLDEN ERA COLLAPSES

| Trigger | Description |
|---------|-------------|
| **Fan Trust Death Spiral** | Scandals, betrayal, lazy booking, or political chaos destroys fan emotional buy-in |
| **Worker Exodus** | Icons leave, factions fracture, new movements steal emotional center away |
| **Cultural Drift Unanswered** | Promotion fails to adapt to world emotional changes (rebellion, realism, escapism): left behind |
| **Leadership Decay** | Booking/ownership instability shatters ability to sustain narratives or protect emotional memory |
| **World Event Shock** | External events (scandals, deaths, real-world crises) break the illusion permanently |

### Collapse Trigger Thresholds

| Trigger | Threshold |
|---------|-----------|
| **Fan Trust** | Trust drops below 30% for 6+ consecutive months |
| **Worker Exodus** | 3+ main eventers leave within 12 months OR ace departs |
| **Cultural Drift** | Cultural alignment score below 40% for 12+ months |
| **Leadership Decay** | Booker trust below 25% OR 3+ regime changes in 24 months |
| **World Event Shock** | Severity 8+ external event directly impacting promotion |

## 2. STAGES OF GOLDEN ERA DEATH (5 STAGES)

| Stage | Name | Description |
|-------|------|-------------|
| **1** | **Emotional Hollowing** | Fans keep watching but stop feeling. Heat for heat's sake. Empty spectacle. |
| **2** | **Event Horizon** | First visible betrayals or scandals fracture mass trust. Hardcore fans begin defecting emotionally. |
| **3** | **Mass Migration** | Fanbase loyalty hemorrhages: to rival promotions, new movements, indie scenes. |
| **4** | **Cultural Mourning** | Hardcore loyalists mythologize "the lost glory days." Emotional nostalgia cements even as live energy dies. |
| **5** | **Legacy Fracture** | Future generations inherit divided memory: golden myth vs collapse trauma. |

## 4. SIMULATION MECHANICS

| Mechanic | Behavior |
|----------|----------|
| **Collapse Memory Flag** | Promotions tagged in world history: "Golden Era (YYYY-YYYY): Collapsed (event/era)" |
| **Fan Memory Fragmentation** | Hardcore vs casual fans carry different emotional memories: tensions bleed into media, fandom culture, worker booking reactions |
| **Worker Legacy Shifts** | Icons during collapse remembered differently based on behavior: stayed loyal? jumped ship? betrayed cause? |
| **Future Booking Penalties** | Promotions with collapse scars must work harder for decades to rebuild fan trust, worker loyalty, world prestige |
| **New Movements Mythologized Faster** | Rival promotions catching emotional migration inherit displaced cultural loyalty at speed |

### Worker Legacy During Collapse

| Behavior | Legacy Effect |
|----------|---------------|
| **Stayed Loyal and Mourned** | Respected, sympathetic, potential Beloved Pillar status |
| **Jumped Ship Early** | Pragmatic but may be resented by loyalists |
| **Stayed Too Long** | Risk of Rotting Legend status |
| **Betrayed Cause** | Blackened memory, potential Reinterpreted Villain |

## 5. EMOTIONAL LAYERS POST-COLLAPSE

| Layer | Fan Behavior |
|-------|--------------|
| **Golden Age Nostalgia** | Hardcore fans mythologize past: keep flame alive in memory even if promotion dies |
| **Angry Betrayal** | Bitter fans blame certain figures (booking, management, wrestlers) for killing the era |
| **Selective Amnesia** | Some fans only remember the boom: erase or deny collapse pain |
| **Ideological Factionalism** | Future fans inherit loyalty splits: some worship old ways, others embrace new forms |

### Nostalgia vs Revisionism Over Time

| Timeframe | Dominant Narrative |
|-----------|-------------------|
| **0-5 years post-collapse** | Angry Betrayal dominant, blame-seeking |
| **5-15 years post-collapse** | Golden Age Nostalgia rises, revisionism begins |
| **15-30 years post-collapse** | Selective Amnesia and Ideological Factionalism compete |
| **30+ years post-collapse** | Mythologization complete, nuance often lost |
