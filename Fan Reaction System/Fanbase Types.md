## Core Fanbase Types

### Hardcore Company Loyalists

**Profile:** Die-hard fans of a specific major promotion. Brand loyalty overrides individual worker attachment.

| Attribute | Value |
|-----------|-------|
| Base Loyalty | 75% |
| Scandal Tolerance | High |
| Core Values | Tradition, brand identity, star worship |
| Turn Triggers | Only "murder-level" scandals; even then, slow to turn |
| Examples | Classic WWE lifers, NJPW purists, old-school NWA territory fans |

**Behavior Patterns:**
- Defend promotion decisions even when unpopular
- Resist criticism from outsiders
- May turn on individual workers but rarely on promotion itself
- Strong nostalgia attachment
- "Company man" workers are favorites

---

### Indie Hardcore Fans

**Profile:** Workrate-focused fans who value authenticity and in-ring skill over mainstream success.

| Attribute | Value |
|-----------|-------|
| Base Loyalty | 30% |
| Scandal Tolerance | Very Low |
| Core Values | Authenticity, workrate, anti-corporate sentiment |
| Turn Triggers | "Selling out," hypocrisy, exposed prejudice, bad matches |
| Examples | PWG faithful, early ROH fans, hardcore puro followers |

**Behavior Patterns:**
- Quick to turn on perceived hypocrites
- Celebrate "discovered them first" pride
- Hostile to corporate wrestling
- Value match quality over character work
- Purity tests are common and ruthless

---

### Family-Friendly Fans

**Profile:** Parents and children who view wrestling as family entertainment. Heroes and role models matter.

| Attribute | Value |
|-----------|-------|
| Base Loyalty | 55% |
| Scandal Tolerance | Very Low |
| Core Values | Wholesomeness, heroism, role models, safe entertainment |
| Turn Triggers | Drug scandals, violence, anything threatening to children |
| Examples | WWE families, Saturday morning cartoon wrestling fans |

**Behavior Patterns:**
- Immediate rejection of scandalous workers
- Strong attachment to clear babyfaces
- Uncomfortable with morally gray storytelling
- Merch-driven (kids demand merchandise)
- Highly influenced by parental gatekeeping

---

### Deathmatch Outlaw Fans

**Profile:** Hardcore fans who value authenticity, toughness, and spectacle over polish.

| Attribute | Value |
|-----------|-------|
| Base Loyalty | 25% |
| Scandal Tolerance | Medium-High |
| Core Values | Authenticity, toughness, blood, spectacle |
| Turn Triggers | Cowardice, inauthenticity, refusing to bleed |
| Examples | CZW faithful, BJW hardcore fans, GCW crowd |

**Behavior Patterns:**
- Tolerate a lot of bad behavior if it feels "real"
- Hate workers who seem scared or soft
- Value sacrifice and pain
- Low promotion loyalty (follow workers, not brands)
- Respect for "outlaws" and misfits

---

### Lucha Traditionalists

**Profile:** Mexican wrestling purists who value tradition, masks, legacy, and local pride.

| Attribute | Value |
|-----------|-------|
| Base Loyalty | 65% |
| Scandal Tolerance | Medium |
| Core Values | Tradition, mask honor, family legacy, Mexican pride |
| Turn Triggers | Mask violations, disrespecting tradition, betraying Mexico for US |
| Examples | Arena Mexico regulars, CMLL purists, lucha families |

**Behavior Patterns:**
- Protect local icons fiercely
- Slower to forgive outsiders than locals
- Mask matches carry immense emotional weight
- Multi-generational loyalty (families attend together)
- Strong regional pride

---

### Joshi Superfans

**Profile:** Dedicated followers of Japanese women's wrestling with idol-culture influence.

| Attribute | Value |
|-----------|-------|
| Base Loyalty | 60% |
| Scandal Tolerance | Low |
| Core Values | Purity, dedication, idol worship, emotional investment |
| Turn Triggers | Sleaze, betrayal, retirement/graduation, scandals |
| Examples | Stardom faithful, classic AJW devotees, idol-fan crossovers |

**Behavior Patterns:**
- Parasocial intensity is high
- Betrayals and sleaze are devastating
- Strong attachment to specific workers over promotions
- Graduation ceremonies are emotional events
- Protective of "their" favorites

---

### Edgy Attitude Era Types

**Profile:** Fans who came up during the Monday Night Wars era. Value antiheroes and controversy.

| Attribute | Value |
|-----------|-------|
| Base Loyalty | 45% |
| Scandal Tolerance | Medium-High |
| Core Values | Antiheroes, controversy, shock value, "badass" energy |
| Turn Triggers | Weakness, corporate kowtowing, boring safety |
| Examples | 1998 Raw viewers, ECW originals, "Attitude Era was peak" crowd |

**Behavior Patterns:**
- Love bad behavior if it's "badass"
- Hate weakness and corporate sanitization
- Nostalgia for edgier era
- May resent modern "PC" wrestling
- Value promos and character over pure workrate

---

### Casual Mainstream Fans

**Profile:** General entertainment consumers who watch wrestling casually.

| Attribute | Value |
|-----------|-------|
| Base Loyalty | 40% |
| Scandal Tolerance | Medium |
| Core Values | Entertainment, big moments, recognizable stars |
| Turn Triggers | Boring product, confusing storylines |
| Examples | WrestleMania-only viewers, lapsed fans, channel surfers |

**Behavior Patterns:**
- Follow media narrative closely
- Susceptible to PR spin
- Drift in and out based on mainstream buzz
- Remember only the biggest stars and moments
- Less investment in "insider" concerns

---

### Smark/Internet Fans

**Profile:** "Smart marks" who follow insider news, critiques, and appreciate meta-awareness.

| Attribute | Value |
|-----------|-------|
| Base Loyalty | 35% |
| Scandal Tolerance | Low-Medium |
| Core Values | Insider knowledge, workrate, booking critique, meta-awareness |
| Turn Triggers | Bad booking, hypocrisy, ignoring their preferences |
| Examples | Reddit/Twitter wrestling communities, podcast listeners, dirtsheet readers |

**Behavior Patterns:**
- Quick to turn on poorly booked favorites
- Appreciate self-aware storytelling
- Critique everything, including themselves
- Can be won back with "smart" booking
- Highly vocal online

## Era Shifts in Fanbase Composition

Fanbase composition changes over time as culture shifts:

| Era | Dominant Fanbase Types | Notes |
|-----|------------------------|-------|
| **Territory Era (pre-1984)** | Casual Mainstream, Family-Friendly | Local, live-event focused |
| **Hulkamania (1984-1992)** | Family-Friendly, Casual Mainstream | Cartoon heroes, mainstream crossover |
| **New Generation (1993-1996)** | Family-Friendly (declining), Smark emerging | Transition period |
| **Attitude Era (1997-2001)** | Edgy Attitude, Casual Mainstream | Peak mainstream, teen male demo |
| **Ruthless Aggression (2002-2007)** | Hardcore Loyalists, Smark/Internet growing | Internet discourse rises |
| **PG Era (2008-2015)** | Family-Friendly, Hardcore Loyalists | Corporate sanitization |
| **Reality Era (2016-2019)** | Smark/Internet, Hardcore Loyalists | Blurred lines, social media |
| **Modern Era (2020+)** | Smark/Internet, Casual Mainstream split | Tribalism, fragmented audiences |
