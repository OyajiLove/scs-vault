A push is a **world-recognized effort to elevate a worker.**

## Push Level Tracking

**Push Level (0-100)** represents how hard the promotion is positioning a worker as a star, independent of whether crowds have bought in.

### Push Level Factors

| Factor | Weight | Description |
|--------|--------|-------------|
| **Card Position** | High | Main event = high push, opener = low push |
| **Win Percentage** | High | Protected winners vs. 50/50 vs. jobbing |
| **Title Proximity** | High | Champion, challenger, contender, or nowhere near gold |
| **TV/Promo Time** | Medium | Segment length, promo opportunities, spotlight moments |
| **Commentary Treatment** | Medium | "The future of this business" vs. "solid hand" |
| **Merchandise Push** | Medium | Featured on posters, ads, merch displays |
| **Opponent Quality** | Medium | Working with main eventers vs. enhancement talent |
| **Booking Protection** | High | Never loses clean, always looks strong |

### Push Level Scale

| Range | Push Tier | Description |
|-------|-----------|-------------|
| 90-100 | **Franchise Player** | "The Guy," everything revolves around them |
| 80-89 | **Top Star Push** | Main event lock, heavily protected |
| 70-79 | **Upper Card Push** | Regular main eventer, title program ready |
| 60-69 | **Strong Midcard** | Building toward upper card |
| 50-59 | **Midcard** | Solid positioning, occasional spotlight |
| 40-49 | **Lower Midcard** | Wins sometimes, loses to people above |
| 30-39 | **Enhancement Plus** | Occasionally wins, mostly builds others |
| 20-29 | **Jobber** | Exists to lose |
| 0-19 | **Invisible** | Barely on TV, no investment |

## 6 Tracking Layers âœ… LOCKED

| Layer | Tracked Through |
|-------|-----------------|
| **Booking Memory** | Who they beat, how often they're featured, what segments they're trusted with |
| **Fan Reaction Memory** | Are crowds buying in? Are pops growing or fading? Is heat earned or forced? |
| **Title Proximity & Match Stakes** | Not just "did they win the belt," but were they near it, chasing it, or used to elevate it? |
| **Emotional Arc Coherence** | Does it feel like a story, or just appearances? Was there payoff? |
| **Commentary & Media Echo** | Did the narrative catch on? Are analysts treating it as real? Or mock-worthy? |
| **Locker Room Perception** | "This guy's getting the rocket." Or: "Another flavor of the week." Resistance, politics, and sabotage emerge from how the push is perceived internally. |

## 6 Push Types âœ… LOCKED

| Type | Description |
|------|-------------|
| **Classic Rise** | Wins, builds crowd momentum, gets signature moments, peaks at the right time |
| **Redemption Push** | Comeback arc: injury return, post-burial surge, veteran reinvention |
| **Crash Push** | Fast-track star whose heat outpaces belief. Often ends in backlash. |
| **Token Push** | Perceived as shallow: checking a box, pleasing a sponsor, or responding to backlash |
| **Faction Push** | Group association elevates someone. Risk: may vanish if faction collapses. |
| **Heir Push** | "The chosen one" story. Can work... or burn the fed down. |

## Internal Emotional States âœ… LOCKED

Used internally to drive fan reaction, backlash risk, push success, and commentary behavior:

| State | Description |
|-------|-------------|
| **Hopeful** | Crowd is rooting for them. Arc feels right. |
| **Pretender** | Getting pushed, but not believed. Crowd disconnect. |
| **Survivor** | Hanging on. Not hot, but trusted. Useful. |
| **Cursed** | Booked poorly for too long. Pushes don't land anymore. |
| **Rocketed** | Sudden elevation. Can be breakout or backlash bait. |
| **Ascendant** | Momentum + trust + story = crowd believes. |
| **Stalled** | Was climbing. Now stuck. Booking unsure. Fans hesitant. |
| **Cemented** | Emotionally confirmed as "real." No more proving needed. |
| **Iconic / Mythbound** | Even if retired, fans treat them like gods. |

**Hybrid Approach:** Public-facing tiers for usability. Internal emotional states track the logic.

## Key Insight

> **Apathy is worse than heat. Boos are engagement. Silence is death.**

A worker in Floundering state isn't hated, they're *nothing*. The crowd has moved on. This is the hardest state to escape.
