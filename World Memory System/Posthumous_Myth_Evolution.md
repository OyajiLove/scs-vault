## 1. MYTH EVOLUTION CAUSES (4 DRIVERS)

| Cause | Description | Direction |
|-------|-------------|-----------|
| **Authentic Tribute Preservation** | Workers whose careers and deaths are honored sincerely see mythological prestige grow steadily over generations | POSITIVE |
| **Scandals Discovered Posthumously** | Dark truths, betrayals, or crimes surfacing after death cause myth reputation to decay sharply | NEGATIVE |
| **Rediscovery Events** | Forgotten matches, promos, feuds, personal struggles unearthed by historians trigger retroactive myth-building or myth-breaking | VARIABLE |
| **Promotion Narrative Control** | Surviving promotions shape public memory, either sanctifying the fallen figure or slowly phasing them out | VARIABLE |

## 3. LEGACY DRIFT ROLLS

Every generation (approximately 10-15 years), workers' historical emotional reputation shifts:

| Roll Trigger | Shift Direction |
|--------------|-----------------|
| **Rediscovery Event** | +15 to +40 (positive) or -10 to -30 (if dark truths found) |
| **Scandal Exposure** | -20 to -75 depending on severity |
| **Tribute Continuation** | +5 to +15 (maintains/grows) |
| **Promotion Neglect** | -5 to -15 (fades) |
| **Cultural Mood Shift** | Variable (-20 to +20) |
| **Documentary Release** | +10 to +30 (typically positive) |

### Legacy Score Over Time

| Generation | Typical Trajectory |
|------------|-------------------|
| **Immediate (0-5 years)** | Emotional peak, raw grief, hagiography |
| **Short-term (5-15 years)** | Settling, some critical reassessment |
| **Medium-term (15-30 years)** | Crystallization, "final" reputation forming |
| **Long-term (30-50 years)** | Historical figure, subject to rediscovery |
| **Very Long-term (50+ years)** | Mythological, simplified narrative |

## 4. FAN MEMORY SPLINTERING

Different fan factions remember fallen workers differently:

| Faction | Memory Bias |
|---------|-------------|
| **Purist Nostalgists** | Preserve golden memory, resist criticism |
| **Workrate Technicians** | Focus on in-ring legacy, separate from personal |
| **Realism Fundamentalists** | Want full truth, resist hagiography |
| **Anarchist Loyalists** | May embrace controversial figures defiantly |
| **Irony Subverters** | Deconstruct myths, find absurdity |

### Splintering Triggers

| Trigger | Splintering Effect |
|---------|-------------------|
| **Posthumous Scandal** | Major split between defenders and condemners |
| **Promotion Exploitation** | Purists vs pragmatists |
| **Rediscovery of Flaws** | Golden memory defenders vs realists |
| **Cultural Shift** | Generational divide in interpretation |

## 5. FAMILY/SURVIVOR ACTIONS

Families and survivors significantly affect myth trajectory:

| Action | Effect |
|--------|--------|
| **Active Participation in Tributes** | +20-30 legitimacy, myth strengthened |
| **Foundation/Scholarship Creation** | +25-40 long-term preservation |
| **Opposition to Exploitation** | Myth protected from corporate damage |
| **Public Criticism** | -15-30 if family disputes legacy |
| **Silence/Withdrawal** | Myth drifts without guidance |
| **Legal Action** | Variable, can protect or complicate |

### Martha Hart Case Study

**Family Action:** Refused WWE involvement in Owen's memory for 20+ years

**Effect:**
- Protected Owen from potential WWE exploitation
- Created "forbidden" mystique around Owen's legacy
- AEW partnership (2022) became major event because of rarity
- Owen's myth strengthened by scarcity of official content

**Lesson:** Family opposition can paradoxically strengthen myth by preventing overexposure.
