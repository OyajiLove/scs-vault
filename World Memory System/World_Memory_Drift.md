## 1. CAUSES OF MEMORY DRIFT (5 DRIVERS)

| Cause | Description |
|-------|-------------|
| **Generational Fan Turnover** | New fans who didn't live through original events reinterpret history through secondhand stories, media, and nostalgia |
| **Promotion Narrative Control** | Surviving promotions rewrite their past in documentaries, Hall of Fame ceremonies, retcons |
| **Worker Survivor Narratives** | Retired wrestlers, autobiographies, shoot interviews reshape public memory |
| **Cultural Mood Swings** | Political, economic, emotional shifts cause old heroes to be seen differently (e.g., anti-corporate sentiment reframing Vince's legacy) |
| **Rediscovery and Re-evaluation** | Fans and historians rediscover matches, arcs, or figures decades later, changing their standing |

---

## 2. DRIFT TYPES (4 PATTERNS)

| Type | Effect | Example |
|------|--------|---------|
| **Rose-Tinted Mythologizing** | Fans idolize Golden Eras and legends more than they realistically deserved at the time | Attitude Era WWF remembered as flawless rebellion |
| **Bitter Revisionism** | Fans (or workers) expose hidden scandals, political manipulations, or betrayals that shatter old myths | Inokiism NJPW acknowledged only after 2010s revival |
| **Selective Forgetting** | Entire eras, workers, or divisions emotionally forgotten if they didn't fit future narratives | WCW 1989-1992 TV excellence erased by Nitro focus |
| **Cult Preservation** | Hardcore fans preserve "true history" against corporate rewriting, sometimes romanticizing old flaws | DVDVR preserving pre-Nitro WCW legacy |

---

## 3. SIMULATION MECHANICS

| Mechanic | Behavior |
|----------|----------|
| **Generational Shift Timers** | Every ~15-20 years, fanbase memory undergoes drift checks |
| **Worker Reputation Re-Rolls** | Minor workers may fade. Legendary workers may grow in myth, or shrink if scandals surface. |
| **Promotion Legacy Drift** | Surviving brands retell or rewrite past eras, creating new fan consensus over time |
| **Rediscovery Events** | Historical "hidden gems" (matches, workers, promotions) occasionally rediscovered, retroactively affecting world memory and loyalty |

### Generational Drift Timeline

| Years Since Era | Dominant Memory Pattern |
|-----------------|------------------------|
| **0-5 years** | Raw memory, emotional reactions still fresh |
| **5-15 years** | Mythologizing begins, edges soften |
| **15-25 years** | Generational turnover, simplified narratives dominate |
| **25-40 years** | Full mythologization or bitter revisionism crystallizes |
| **40+ years** | History becomes legend, nuance largely lost |

## 4. FRACTURED WORLD MEMORY

Different fan groups remember the same promotion/era differently:

| Group | Memory Style | Example (WCW) |
|-------|-------------|---------------|
| **Mainstream Casual Fans** | Simplified bullet points | Remember Nitro, NWO, Goldberg, "WCW died" |
| **Serious Historians/Hardcore** | Granular eras | Remember 1989-92 TV excellence, Dangerous Alliance, Steiners |
| **Younger Fans Post-Death** | Meme narratives | "David Arquette, Russo, Fingerpoke = LOL WCW" |
| **Regional Loyalists** | Pre-corporate legacy | Remember Crockett, "real wrestling before McMahonism" |

### Why Mainstream Memory Flattens

| Cause | Result |
|-------|--------|
| **Narrative Efficiency** | Mainstream flattens for simplicity ("Rise, Fall, Meme") |
| **Control by Survivors** | WWE shaped post-WCW narrative through documentaries, HOF |
| **Attention Span Decay** | Newer generations prefer short, loud emotional landmarks |
| **Media Laziness** | Surface-level articles reinforce simplified version |

### WCW Layered Emotional Histories (Example)

| Era | Hardcore Fan Memory | Mainstream Memory |
|-----|---------------------|-------------------|
| **1989-1992** | "Best TV wrestling ever. Dangerous Alliance. Steiners." | (Forgotten) |
| **1993-1995** | "Painful but weirdly compelling. Hidden matches." | (Forgotten) |
| **1996-1998** | "Incredible buzz, insane cards, generational excitement." | "NWO! Monday Night Wars!" |
| **1999-2001** | "Crushing sadness. Politics killing what felt real." | "Fingerpoke! Arquette! LOL dead" |

## 8. MEMORY FACTIONS (5 TYPES)

As promotions, eras, and workers age, different fan groups attach to different memories:

| Faction | Emotional Anchor |
|---------|------------------|
| **Golden Nostalgists** | Mythologize early boom eras (80s NWA, 90s King's Road, early AEW) |
| **Punk Rock Purists** | Remember indie revolutions, outlaw feds, workrate booms with romantic reverence |
| **Crash TV Loyalists** | Remember late 90s/early 2000s spectacle and chaos as true soul of wrestling |
| **Workrate Intellectuals** | Obsess over match quality, workrate arcs, technical evolution (DVDVR yearbooks) |
| **Anarchist Outlaw Fans** | Idolize hardcore, deathmatch, and outsider wrestling forgotten by corporate histories |

### Faction Formation Mechanics

| Mechanic | Behavior |
|----------|----------|
| **Factional Loyalty Formation** | As promotions/eras/workers age, different fan groups attach to different memories |
| **Narrative Drift Weighting** | Mainstream narrative gains inertia, but hardcore factions resist, correct, rediscover over time |
| **Rediscovery Events** | Minor memory factions periodically trigger rediscovery arcs (documentaries, fan projects, tribute shows) |
| **Historical Status Re-Roll** | Career/promotion reputation drifts based on which memory faction becomes dominant |

### Memory Split Examples

| Subject | Memory Split |
|---------|-------------|
| **WCW** | Nitro wars idolized by mainstream; early 90s TV golden era preserved by hardcore forums like DVDVR |
| **ECW** | Mainstream remembers "hardcore blood"; hardcore fans remember Raven vs Dreamer emotional storytelling and Masato Tanaka classics |
| **NOAH King's Road** | Hardcore puro nerds mythologize golden arcs; casual fans only remember Misawa's tragic death |

## 9. ATTITUDE ERA CASE STUDY (MEMORY DRIFT IN ACTION)

The WWF Attitude Era demonstrates how memory gets sanitized, fractured, and rewritten over time.

### The Real Sequence (Not the Sanitized Version)

| Phase | What Actually Happened |
|-------|------------------------|
| **Late 1990s (True Attitude Era)** | Genuinely rebellious: Austin vs McMahon, DX shock value, chaotic TV structure. Sometimes organic, sometimes tasteless, but felt alive. |
| **2001-2003 (Post-WCW Shock Value)** | Without competition, WWF doubled down on grotesque elements (Katie Vick, necrophilia jokes, HLA). Not rebellion but pure bad taste. |
| **2003-2008 (Transitional Drift)** | After advertiser pressure, post-9/11 media scrutiny, and audience fatigue, WWE slowly sanitized. Ruthless Aggression still flirted with edge but corporate filtering increased. |
| **2008-Present (PG Corporate Era)** | Full corporate transition. Attitude nostalgia selectively embraced (Rock promos, Austin moments), but genuinely offensive elements buried or never mentioned. |

### Memory Faction Biases

| Faction | Memory Bias |
|---------|-------------|
| **Casual Fans** | Remember fun chaos (Austin beer baths, Rock catchphrases, Socko). Forget vile segments unless reminded. |
| **Hardcore Fans** | Split: some remember wild creativity lovingly, some remember racism/sexism/homophobia with bitterness. |
| **Modern WWE Narrative** | Highlights Attitude Era selectively. Mythologizes "edginess." Ignores Lawler slurs, Mark Henry-Mae Young, worst excesses. |
| **Historians/Analysts** | Increasingly reframe Attitude Era as creatively vibrant but morally toxic: mixture of admiration and deep critique. |

### SCS Model for Sanitization

| Memory Layer | Behavior |
|--------------|----------|
| **Mainstream Memory** | Sanitizes grotesque history, focuses on success and positive myth-building |
| **Hardcore Memory** | Preserves full complexity: brilliance, horror, betrayal, tragedy, creativity intertwined |
| **Rediscovery/Revisionism** | Depending on cultural mood, new fans mythologize harder OR brutally re-examine past |
| **Worker Scandal Fallout** | Figures who seemed like icons can have memory reevaluated decades later if new scandals or cultural shifts occur (Austin, Lawler, Vince) |

## 11. PHILOSOPHY

> "History is not what happened. History is what is remembered. And memory drifts like smoke: shaped by survivors, simplified by time, preserved by the devoted few who refuse to let truth die quietly."

Wrestling history is particularly vulnerable to drift because:
- Surviving promotions control the narrative
- Emotional investment distorts memory
- Generational turnover is constant
- Physical media was poorly preserved
- Shoot interviews reshape perception

SCS models this: not a single "official history" but fractured, competing emotional memories that evolve over decades.
