## 1. BETRAYAL CAUSES (4 FACTORS)

| Cause | Description |
|-------|-------------|
| **Promotion Emotional Drift** | Over time, surviving promotions drift from the ideals the fallen martyrs symbolized, risking emotional betrayal perception |
| **Corporate Exploitation** | Using martyr images, names, titles for cynical cash grabs or shallow marketing. Fans smell the betrayal instantly. |
| **Legacy Disrespect Booking** | Booking arcs that undermine, erase, or humiliate the emotional memory of the dead. Shatters fan trust violently. |
| **Worker Disavowals** | Survivors breaking public loyalty to martyrs, either from burnout, bitterness, or corporate pressure. Triggers emotional rage events. |

---

## 2. BETRAYAL FALLOUT TYPES (4 FORMS)

| Betrayal Type | Emotional Reaction |
|---------------|-------------------|
| **Emotional Fan Rebellions** | Hardcore loyalists turn violently against promotions. Loyalty collapses, mass migration to rivals or indie scenes. |
| **Historical Memory Drift** | Martyr myths mutate. Betrayed martyrs become even more romanticized by underground fans, while official histories rot. |
| **Worker Emotional Exoduses** | Workers emotionally tied to martyrs defect, rebel, or silently disengage. Locker room cultural collapse accelerates. |
| **Promotion Scandal Stains** | Betrayal of martyrs locks permanent loyalty penalties. Even generations later, promotions carry scars unless redemption arcs occur. |

### Fallout Severity Matrix

| Betrayal Type | Immediate Damage | Long-term Damage |
|---------------|------------------|------------------|
| **Corporate Exploitation (mild)** | -10 to -20 loyalty | Slow erosion over years |
| **Corporate Exploitation (severe)** | -30 to -50 loyalty | Permanent scar |
| **Legacy Disrespect Booking** | -40 to -60 loyalty | Catastrophic, near-permanent |
| **Worker Disavowal (lone wolf)** | -15 to -25 loyalty | Worker reputation damage |
| **Worker Disavowal (group)** | -30 to -40 loyalty | Promotion crisis |

## 3. SIMULATION MECHANICS

| Mechanic | Behavior |
|----------|----------|
| **Hidden Martyr Loyalty Anchors** | Promotions carry invisible emotional weights tied to fallen legends. Loyalty boosts while respected, loyalty collapses if betrayed |
| **Betrayal Event Triggers** | Booking humiliations, title disrespect, marketing abuses, or public disavowals trigger Martyr Betrayal Events |
| **Fanbase Split and Migration** | Betrayal splits fanbases: Purists abandon, Pragmatists drift, Cynics mourn but stay detached |
| **World Memory Scar Permanence** | Martyr betrayals log permanent emotional scars into world memory. Cannot be erased, only healed over decades (if ever) |

### Fanbase Split by Type

| Fan Type | Response to Betrayal |
|----------|----------------------|
| **Purists** | Abandon immediately. Will not return without major redemption. |
| **Pragmatists** | Drift away gradually. May return if promotion improves. |
| **Cynics** | Mourn but stay detached. Already expected betrayal. Low loyalty but stable. |
| **Casuals** | May not notice. Minimal impact unless media coverage amplifies. |

### Scar Permanence Timeline

| Betrayal Severity | Minimum Healing Time |
|-------------------|----------------------|
| **Minor (tasteless marketing)** | 2-5 years |
| **Moderate (disrespectful booking)** | 5-10 years |
| **Severe (legacy humiliation)** | 10-20 years |
| **Catastrophic (complete exploitation)** | Permanent (partial healing only) |

## 4. HISTORICAL MODELS

| Event | Martyr Betrayal Memory |
|-------|------------------------|
| **WWE Exploitation of Eddie Guerrero Memory (2005-06)** | Immediate emotional backlash among hardcore fans. Loyalty erosion long-term. Rey title run framing felt exploitative to many. |
| **AJPW Mishandling of King's Road Memory (Post-2001)** | Fans emotionally detached. Only NOAH and indie purists kept emotional loyalty alive for Misawa/early 90s spirit. |
| **WWECW Betrayal** | Corporate hollow remake. Emotional death of the original ECW myth among most surviving loyalists. Model case of catastrophic exploitation. |

### WWECW Case Study

**Context:** WWE revives ECW brand (2006)

**Betrayal Type:** Corporate Hollow Remake

**What Made It Betrayal:**
- ECW's soul was anti-corporate rebellion
- WWE repackaged it as sanitized nostalgia product
- Original ECW talent reduced to cameos
- New ECW had none of the original philosophy
- Paul Heyman eventually walked away

**Fan Response:**
- Purists: Rejected immediately, called it "WWECW" mockingly
- Pragmatists: Tried it, drifted away
- Original ECW loyalists: Felt personally betrayed
- Casual WWE fans: Confused by references to "old ECW"

**Long-term Effect:** Original ECW myth strengthened by contrast. WWECW remembered as cautionary tale. Permanent stain on WWE's handling of acquired properties.

**Lesson:** You cannot corporatize rebellion. The attempt itself is the betrayal.
